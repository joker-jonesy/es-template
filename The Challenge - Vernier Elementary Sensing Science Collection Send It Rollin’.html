The Challenge - Vernier Elementary Sensing Science Collection: Send It Rollin’

The Challenge

Two vehicles are rolling downhill; one is fully loaded and the other is empty. Which one is going to get to the bottom of the hill first? Your challenge is to create a downhill ramp using the materials provided and roll a toy vehicle down the ramp while adding more weight each time.  Using the Vernier Go Direct Motion Detector, you will measure the speed of a toy vehicle as it goes down the ramp.  

What I'll Learn 

  • I CAN create a hypothesis based on a real-world situation.
  • I CAN use collected data to create a scaled line graph.
  • I CAN determine the difference in how speed is affected by weight.
  • I CAN evaluate my hypothesis to determine if I was correct or incorrect.

Stuff I'll Need

  • Vernier Elementary Sensing Science Collection
  • Computer, iPad, or Tablet with Graphical Analysis 4 software 
  • Graphical Analysis Instructional Manual
  • Go Direct® Motion Instruction Manual
  • Send It Rollin’ Data Sheet
  • Vernier Elementary Sensing Science Accessory Collection
  • Weights (rolls of pennies, D batteries, etc.)
  • Materials (to create ramp - books, sturdy cardboard, tape, toy truck/car, etc.)
