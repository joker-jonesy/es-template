Extend Yourself - Sphero: Germs Everywhere

Social Distance

Can you program your robots to social distance? Determine the distance they need to keep to stay healthy. Program your robots to stop or move away from one another when they reach this distance from one another.

The Cure

Add another BOLT robot to the group. Program this BOLT to be the cure. You will need to update the previous two programs you created (Infection and Healthy). Keep in mind this BOLT will be immune to the infection. Use the following criteria to spread the cure to the other BOLT robots that have been infected:

  • The cure BOLT will need to send out IR messages with the cure continuously.
  • Decide which animation you would like to play as BOLT randomly moves around, curing its friends.
  • BOLT will need to roll around like the infected robots, but just a bit faster.
  • Use channel 5 and randomize the intensity to help spread the cure.

Try to update the program of the BOLT that spreads the infection. It needs to be able to be cured as well. What changes will you need to make to that program?

Finally, as you run your trials, fill out a table and line plot to track how quickly the cure spreads.
