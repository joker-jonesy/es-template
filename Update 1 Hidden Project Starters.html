Minimize the Scroll

With all the exciting project starters available to your students, we understand it can be a lot for a student to sort through. We have made updates that allow facilitators to develop a focused learning path that allows students to see just the content they need for each rotation. You can now plan a rotation and customize your students' LearningHub view accordingly. 

The Details

  • By default, all technologies and project starters are hidden from student view. 
  • Facilitators will still be able to view and access all project starters. 
  • You now have the ability to turn off / on student visibility for technologies or project starters. 

The Plan

  • Learn how to modify page visibility by watching this video
  • Plan the technologies and/or project starters you will use in your first rotation. For assistance, use the provided rotation maps.
  • Go into your course and turn on visibility for anything you want students to see for rotation one.
  • After students complete rotation one, turn off the visibility of the technologies and project starters. 
  • Repeat the process above for all future rotations.
