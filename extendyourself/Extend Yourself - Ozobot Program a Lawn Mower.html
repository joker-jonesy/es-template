Extend Yourself - Ozobot: Mow a Lawn

 From Garage to Lawn

What if the user wants the lawn mower to leave the garage, drive to the lawn, and then start mowing? How could you change the code to make this happen?

  1. Try programming Ozobot to drive to the lawn using the sidewalks.
  2. Next, try making the sidewalks curvy. Can Ozobot still find the lawn?
  3. What if a toddler decides to sit on the lawn? Can you program Ozobot to mow around the child?
  4. When you code, remember to make sure Ozobot knows not to drive into buildings and things that can break it. Also, make sure it does not drive through and ruin any gardens.

 Maze Exchange

Create a maze, and then use OzoBlockly to program Ozobot to win.

  1. Design a track or a maze. You can use different colored lines, but do not use colored OzoCodes. Use OzoBlockly for coding.
  2. Write the code in OzoBlockly that will get Ozobot to the finish.
  3. Bonus Challenge: Trade your maze with another group. See if they can figure out the code to get Ozobot to the finish.

 Learn About Automated Guided Vehicles (AGV)

Ozobot can be an Automated Guided Vehicle (AGV). Can you figure out why? This image shows AVGs at work. How is Ozobot like the AGVs seen to the left?

Research:

  1. Who invented the first AGV and why?
  2. What are AGVs used for?
  3. What kinds of AGVs are there?
  4. What industries use AGVs?
  5. How could an AGV help you? Your family and friends? Your school?

 AGV Design Challenge

The manufacturers of your favorite game are having trouble keeping up with the demand. AVGs could help, but they would need to work together and cross paths without getting in each other’s way. Can you design a workflow for the AGVs?

Use Ozobot to simulate the process of moving the games from the parts room to assembly, and then to packaging.

You can partner with other groups so you have multiple Ozobots to work with, or you can plot out their paths and timings and show how they could operate without crashing or missing their loads.

 Program Your Robot to Mow ANY Lawn

Now change your program so it can figure out how to mow different lawns.

Preparing for Your Advanced Simulation:

  1. Create a lawn in any shape.
  2. Create more sidewalks, gardens, etc., or reuse the ones you had from before to design a new yard.
  3. While Ozobot mows, it should light up green.
  4. If Ozobot drives on the sidewalks or driveway, it should light up red to show that it is not mowing.
  5. When you write the code, make sure Ozobot knows not to drive into buildings and things that can break it. Also, make sure it does not drive through and ruin any gardens.
  6. Remember to take video and screenshots, and record your observations.

You can use toolbox 2 to complete this task, but you may want to use toolbox 3 or 4 since they have Logic Statements.
