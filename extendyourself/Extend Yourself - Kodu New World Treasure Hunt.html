Double The Area

Create another world that is double the area. How does the perimeter change? If another world is created that is double the perimeter, how would the area change? How would you double the number of hills, rocks, lakes, or any other feature you included in your original design? Evaluate each of these by using expressions with parentheses or brackets.

Create a Game

Create a game, using paper and pencil or a different app, where players evaluate expressions using parentheses and brackets to find their treasure. As an example, you could draw a game board similar to the one below and fill in each block with a different expression that uses parentheses and/or brackets. Once the board is filled out, play the game with a partner or small group. The first person could roll a number cube and move a certain number of spaces. That person must correctly evaluate the expression to stay on that spot. Otherwise, he or she would have to return to their previous location. The first person to the end wins.
