Extend Yourself

  • Learn how to share your Do Ink animation projects.
  • Use the trace tool and layers to create a drawing based on a picture you like. Watch the How to Use Trace Feature on this page to learn how.
  • Make your animation complex by adding several more drawings. You can add static drawings, which means they consist of one frame and don't move, and animated drawings that consist of multiple frames. Add these drawings in at different times. Make them each have their own unique motion path.
  • Create an animation of a process or event that causes something to change, like how erosion changes the way land is formed. Add text to explain the process. Use the #3-How to Add Text tutorial on this page to learn how.
