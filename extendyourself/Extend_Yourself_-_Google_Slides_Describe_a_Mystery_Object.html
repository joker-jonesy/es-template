Mystery Number

An object can have a certain smell, texture, and can even make a certain sound. But what about a number? Create a presentation about a mystery number. Instead of using senses to describe, you have to find different methods to describe it. For example, the mystery number is a single digit between 3and 9. The mystery number is a factor of 2.

Images Only

Can you get information across only using images? Choose an ecosystem (jungle, ocean, pond, forest, arctic, etc.) and make a presentation about the plants, animals, and weather that occurs in that ecosystem by only using pictures.
