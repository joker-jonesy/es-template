Find Another Graph

Find another type of graph that could be used to record numerical data. See if you can change one of your line plots into the new type of graph. Some ideas are a bar graph, where each value has its own bar, or a histogram, where values are chunked together into equal sized intervals. You can go to Make a Bar Graph to get started.

In this bar graph, the horizontal axis represents the number of volts and the vertical axis represents the frequency (how many times) each volt reading was recorded. Each value has its own bar.

The following histogram represents the same set of data. Instead of each bar representing one value for the volts, each bar represents a range of values. For example, the first bar includes all the values greater than 0 but less than 2. The bar graph has 2 bars representing 3 values in that range. The histogram groups both bars together, but also shows 3 values.

Which type of graph makes it easiest to understand your data? If you’re unsure, ask your classmates. How did the graph make it easier to understand the data?
