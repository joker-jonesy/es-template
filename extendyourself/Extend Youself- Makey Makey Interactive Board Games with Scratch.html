Extend Yourself - Makey Makey: Board Games and Fractions

  • Add more special events and create a Scratch program for each one. Use the letter pins on the back of the Makey Makey circuit board for more inputs.
  • Think about how adding or subtracting mixed numbers is different from adding or subtracting fractions with unlike denominators. Add a few new cards to the deck that include this skill.
  • Create a card game based on the game “War”: Write 10-15 additional fraction equations on index cards. Split the cards with a classmate. Turn over one index card at the same time as your classmate. Solve the equation. Compare your answer with your classmate's answer. Whoever has the largest fraction as an answer wins the round and takes both cards. The game ends when one person has all of the cards.
