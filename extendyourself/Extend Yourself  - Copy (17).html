Extend Yourself 

  • Write your own code language. What symbols will you use? How will you make your commands clear and short? What will you call your new code language? Rewrite the code for your favorite challenge in your new code and share it with a classmate. 
  • Code Combat teaches a third programming language called Coffee Script. Coffee Script is an experimental programming language that is similar to Java Script. Try solving your favorite challenge with Coffee Script. What do you like/dislike about this programming language? How is it similar and different from Java Script. Record your ideas in your ePortfolio. 
  • HTML and CSS text based code along with many other important skill are taught in higher levels of Code Combat. Research HTML and AI scripting and pick a few reasons why it could be helpful for to you and your classmates. Use the research you have gathered to help you write a letter to your facilitator and principal purposing that they fund the Code Combat subscription for your school.
