Extend Yourself - Tinkercad Circuits: Piano Circuit

Compose Music

Compose or write a song using the notes on your piano circuit. Give each one an name and then write the song notes using the names. See if a classmate can read your music and play your song using your piano circuit.

Create an Alarm

Use your new breadboard skills and add components to create some type of an alarm. Try using a different power supply and see if anything changes.
