 Print Your Design

Do you have a 3D printer in your SmartLab? If so, ask your facilitator to print your 3D object. Every 3D printer works a little differently, so make sure you understand how to use your SmartLab printer.

Rule: Only print objects that YOU designed. If you download someone else's design, change it to make it your own.

 3D Printing Useful Objects

3D printers can be used to make all kinds of useful objects. What kinds of uses for 3D printing can you think of?

Design a new invention using Tinkercad that you can 3D print.

Ask yourself:

  • What problem can I solve with my invention?
  • How will my invention work?
  • Who will use my invention? Will they need instructions?

Share your invention with someone else. Ask them how they would improve it.

Printing Real-Life

Select a real-world object that you’d like to print using a 3D printer. To keep it in proportion with the real-world object you want to print, you’ll need to determine how to reduce it so that it looks the same when you print it. Take different measures of your object, and then apply a scale factor to reduce each measure. Design your object on Tinkercad. Does it look the same?
