Extend Yourself - Code Combat: Introduction to JavaScript

Create Your Own Code

Write your own code language that incorporates mathematical symbols. What symbols will you use? How will you make your commands clear and short? What will you call your new code language? Rewrite the code for your favorite challenge in your new code and share it with a classmate.

Explore CoffeeScript

Code Combat teaches a third programming language called CoffeeScript. CoffeeScript is an experimental programming language that is similar to JavaScript. Try solving your favorite challenge with CoffeeScript. What do you like/dislike about this programming language? How is it similar and different from JavaScript?

Research HTML and AI Scripting

HTML and CSS text-based code along with many other important skills are taught in higher levels of Code Combat. Research HTML and AI scripting and pick a few reasons why it could be helpful for you and your classmates. Use the research you have gathered to help you write a letter to your facilitator and principal proposing that they fund the Code Combat subscription for your school.

Investigate Fractions in Everyday Life

Investigate the use of fractions and decimals in everyday life. Here is a great place to start. Take what you learn and create a fun poster to share with your classmates on the real-life uses of fractions and decimals.
