What is Kodu?

Kodu is a computer program you can use to create your own games.

With Kodu you can:

  • Create computer games
  • Build your own game worlds
  • Share games with other Kodu users
  • Play games made by other Kodu developers

How Do You Use Kodu?

Getting Started

  1. Open Kodu Game Lab
  2. Login - You can sign in as a guest or with a creator name and pin number.
  3. Select New World to start creating your game.

Edit Mode

Your screen will look similar to the image below when you are in edit mode.

Toolbar

You will use the toolbar at the bottom of the screen to create your game.

Select each item to learn more.

The home button will take you back to the home menu.

The play button will take you out of the edit mode for your game and allow you to test or play the game you have designed.

The hand tool will move the camera and change your view.

The object tool is where you can find characters and objects to add to your game.

The path tool allows you to program objects to move along a path.

Use the landscape tools to:

  • Add and customize terrain
  • Add hills, valleys, spikes and water

Use the erase tool to delete items from your game.

Use this tool to change the setting in your game.

Short-Cuts

Short Cuts will appear in the top, left corner of the screen.  This is a great place to look if you need help with your game. 

Game Storage

The temperature gauge on the right side of your screen reflects how much storage you have for your game.

Add Terrain

Learn how to use paint tools to create your own world.

To get started, watch the “Add Terrain" video. Learn how to add terrain to your level using the keyboard and mouse.

Add Objects/Characters

  1. Click on the Objects Tool.
  2. Click on the terrain in your game to get the Objects Wheel.
  3. Choose the object you want to add to your game.

Program your Game

You can write lines of code for most of the objects in your game.  Start by programming your game character to move.

Select each item to learn more.

Step 1: Find Object or Character

Click the Objects Tool.

Step 2: Choose Object or Character

Right click on the object/character you want to program.

Step 3: Start Program

Left click on Program.

Step 4: Keyboard and Arrows

Click the plus sign next to WHEN to select keyboard and arrows.

Step 5: Move

Click the plus sign next to DO to select move.

Step 6: Test

Test your code!  Press the ESC key twice to make sure that your character moves when you use the arrow keys on your keyboard.

Changing Settings

Learn how to change the settings of different objects. Set them to move faster or slower, and to see or hear events at different distances. With the Object tool selected, you can Right-Click an object if you want to change its size, height, or other settings.

Click Change Settings and explore.

How to Save Your Work

  1. Click the Configure Kodu icon. This will open the Kodu Settings menu.
  2. Next, choose the location where you will save your work. Click the button under Save Folder and browse to your student folder.
  3. Click OK.
  4. Important: Do this EVERY day before you start using Kodu.

Key Programming/Kodu Concepts

  • Object – Objects are anything you put in your world like a person, place, or thing.
  • When – this is the event that makes something happen.
  • Do – this is the behavior or reaction to the event.
