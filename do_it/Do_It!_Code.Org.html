Do It! Code.org: Hour of Code

 The Challenge

You will write basic computer code as you solve 20 fun puzzle games in the Hour of Code challenge.

 Project Steps

  1. Do It!

  2. Need More Time?

  3. Get Your Certificate

Do It!

Start the Hour of Code website and follow along with the video tutorials and programming “puzzles”.

Need More Time?

Do not worry if you cannot complete all 20 puzzles during the class. You can always go back to the Hour of Code website and finish later. Use the puzzle counter at the top of the page to pick up where you left off.

Get Your Certificate

After you have finished the 20 puzzles you can fill out a Certificate of Completion that will be emailed to you. If you cannot email the certificate, you can take a screenshot of your final puzzle for your project submission.
