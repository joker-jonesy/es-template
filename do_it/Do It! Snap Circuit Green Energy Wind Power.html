Do It! Snap Circuits Green Energy: Wind Power Data Collection

 The Challenge

Your challenge is to learn about alternative energy systems and wind power. Then build your own wind power circuits! You will keep track of the voltage your windmill generates in different positions and record the data on a line plot. Then you will use the line plot to analyze your results.

Create Your Line Plot

Use a table to gather the data while you conduct your tests for each project. Use a tally mark for the number of times you get each result on the voltmeter.

For each of the projects you complete, create a line plot to record the number of volts generated by the windmill in different positions. Make sure each line plot has a title so that you know what project it goes with. Since you will be plotting the results of the voltmeter, your line plot should be set up so that the values are the same as those on the meter. Here is a template of how to set up a line plot for Project #57.

When you have finished the project and recorded your results, write one or two statements to analyze the data. For example:

  • What is the difference between the highest and lowest values you recorded?
  • Is the data grouped around a certain value?
  • What is the sum of all the data values?
  • What is the average data value?
  • Which value occurred the most? How many times did it occur?
  • Which value occurred the least? How many times did it occur?
  • Compare your line plots to the results of another group. How are they similar? How are they different? Why do you think the differences occurred?
