Wiggle and Flip Multiplication!

Your challenge is to use Wiggledoodle-ish and Flipbook-ish to model the Commutative and Associative properties of multiplication. 

By modeling both properties, you will show a situation when using each property can help you multiply. The Commutative Property is useful when you want to multiply in a different order. You might choose a different order that is easier to multiple based on the math facts you know.

For example, you need to multiply 8×2×4. 8×2 = 16. 16×4 is a math fact you may not know. Instead, multiply 2×4 = 8, and then multiply 8×8, which is a math fact more people know.

The Associative Property is useful when multiplying many numbers. Grouping lets you make smaller or more familiar parts to multiply.

Part 1 

  1. Use Flipbook-ish. Make an animation to model the Commutative Property of Multiplication. The example shows six groups of four (6×4) and four groups of six (4×6).
  2. Start with several objects in one big group. Animate the objects to show them being placed into groups with the same number of objects in each group. For example, if you start with 18 objects, you may animate them to show 3 groups of 6 or 2 groups of 9. Be sure that when you count your objects you can place the same number of objects into each group you make.
  3. Write the multiplication sentence.
  4. Reanimate your objects to show a second related multiplication sentence using the commutative property of multiplication.
  5. Write the related multiplication sentence.

Part 2 

  1. Using Flipbook-ish, make an animation to model the Associative Property of Multiplication. The example shows 4 groups of 3, two times, which is (4×3)×2, and four sets of 3 groups of 2, which is 4×(3×2).
  2. Start by placing several objects into groups. For example, you may start with 30 objects as 2 groups of 5, three times (2×5)×3.
  3. Write the multiplication sentence.
  4. Animate your objects to show a second related multiplication sentence using the associative property of multiplication.
  5. Write the related multiplication sentence.
  6. Make videos of your animations to upload for your project submission.
