Do It! UBTECH: Walk the Line

 The Challenge

Your challenge will be to build a robotic car that is able to move forward.  You will use your measuring skills to measure the distances the car travels and then will create a line plot of your data. You will share pictures of your robot, your code, and the data collected for your project submission.

 Project Steps

  1. Build Your Robot

  2. Code Your Robot

  3. Measure Your Distance

  4. Create a Line Plot  

Build Your Robot

You are ready to build your robotic car. This is a custom design and is not included in your UBTECH UKIT EDU App. Follow the directions found here.

Code Your Robot

Now that your robot is built, it’s time to code your robot. Upload the UBTECH UKIT EDU App to your phone if you have not already done that. 

Step 1: Select My Project, then select the plus sign to add a new project. 

Step 2: Turn on your robotic car and click the Bluetooth icon to connect. The app will search for your robot. Make sure it is turned on!

Step 3: Select your robot from the list. If there is more than one robot listed, choose the one with the strongest signal strength. 

Measure Your Distance

Using the program you just created, measure the distance the robot car travels. Record the distance in the data table below. Start the robotic car at the zero point of your ruler. Line up the front axle with the 0 on the ruler. 

The program tells the robot car to move in a straight line. You may see the robot  move a little bit as it finds the correct orientation. The 4 second wait that you programmed into the robot gives you time to make sure the robot is lined up with the ruler correctly before it begins to move.

Once you have measured the distance the robot traveled, program the robot to move twice as far.  Then, program the robot to travel twice as far.  This may take a few tries and that’s okay! When you have successfully gotten the robot to travel twice as far, take a screenshot of the program you altered and insert it into the table for your project submission. If there is time, program the robot to go half as far or three times as far.

Create a Line Plot

Now, you will add to the original program. Return to the presentation - start at Slide #17

You will be adding to the original program a random number generator. This program will randomly generate a distance that the car will travel without you knowing what that distance will be ahead of time. Run the robot ten times. Each time the robot runs, measure how far it traveled and record it in the data table below. When you have ten measurements, organize them by creating a line plot. You will upload this in your Final Project Submission.
