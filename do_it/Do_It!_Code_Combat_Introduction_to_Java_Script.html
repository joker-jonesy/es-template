Do It! Code Combat: Introduction to JavaScript

 The Challenge

Your challenge is to use JavaScript coding language to program your hero in Code Combat to defeat evil and collect gems. Then, you will use symbols found in JavaScript language (<, >, =) to compare decimals to the thousandths place.

 Project Steps

  1. Start Coding With Code Combat

  2. Tally the Code You Use

  3. Most Common Code

Start Coding With Code Combat

  1. Go to the Code Combat website. Click "Try The Game".
  2. When you open the first checkpoint Dungeons of Kithgard you will need to choose your hero and pick the JavaScript programming language.
    1. If you have to reload the page you must double-check that you have JavaScript selected as the programming language.
    2. You can double-check the correct programming language is selected at any time by looking at the right-hand corner of your game screen.
    3. If you need to change the programming language, go to “Game Menu” at the top and select Change Hero. You can choose your programming language at the bottom.
  3. Read the next step, Tally the Code You Use, and then start coding!
  4. As you go, take screenshots of some of the more difficult coding challenges. If you struggle with one of the challenges, take a screenshot of your original code and of the changes you made to try to succeed in the challenge.
