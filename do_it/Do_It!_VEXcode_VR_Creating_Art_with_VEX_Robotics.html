Do It! VEXcode VR: Shape Art

 The Challenge

Your challenge is to program your VEX robot to create a tessellation.

Robots need step-by-step instructions to do a job. They cannot do anything on their own; they can only do what humans program them to do. You will first explore why some shapes fit together while others do not. Then you will practice how to program the VEX robot to create shapes. Finally, you will put all these skills together to create tessellations. This will get you ready to think about why some shapes fit together and others do not.

 Project Steps

  1. Open the Playground

  2. Plan and Sketch Your Tessellation

  3. Program Your Robot

  4. Test and Debug

Open the Playground

The Playground lets you see a simulation of your program in action. 

  1. Open VEXcode VR.
  2. Open the Playground simulator.
  3. Open the Art Canvas playground.

Plan and Sketch Your Tessellation

Before programming your robot, plan your tessellation. Use grid paper for this step. For triangles, use a special kind of grid paper called isometric. You can download isometric grid paper here.

What shapes will make up your tessellation? You may want to start with just one shape.

  • Can you use any shape?
  • Will you need to use rotations or reflections?
  • Will different categories of quadrilaterals fit together without gaps?

Rectangles are easy because they have square corners. Remember that squares are rectangles! But what about quadrilaterals that are not rectangles?

Be sure to think about the properties of the shape that you want to create. How many sides does it have? Does it have square corners? Does it have sides that are the same length?

How will your robot turn to create the angles and sides of each shape? How will each shape connect or overlap with the other shapes next to it?

Once you have planned your design, describe the properties of the shape you chose. Explain why you chose that shape.

Program Your Robot

Now you will use the code blocks to control the pen and draw your tessellation. 

Drag blocks into the scripts area to form your program.

Controlling the Pen

Use the purple code blocks in the Looks section to set the robot pen color and move the pen down so it can write on the canvas.

Creating a Shape

Use blue code blocks in the Drivetrain section to move the robot forward, backward, and turn.

Did your code result in a polygon? If not, can you figure out how to make the robot ‘close the shape’?

Creating a Tessellation

Use the orange repeat, or loop, code blocks in the Control section to repeat code blocks to create multiple shapes. Describe this tessellation with a multiplication sentence.

Test and Debug

Once you have written your initial program, it is time to test it out.

You can run your program by clicking on the play button inside the Playground simulator.

Sometimes your program doesn’t work the way you intended. Debugging, or finding mistakes and fixing them, is a normal part of software engineering.

If your program doesn’t work at first, go through and find where you may need to add, change, or remove code blocks in your program.

Reset the playground simulator and run your program again. Continue this process until your robot creates your tessellation.

Once you have a tessellation with one shape, plan a tessellation with two shapes. If you have time, program your robot to create this new tessellation. Be sure to describe the properties of each shape.
