Do It! Makey Makey:  Board Games and Fractions

 The Challenge

Your challenge is to design a board game that includes a circuit, and requires the player to add and subtract fractions with unlike denominators in order to make a move on the board. Then, use Makey Makey and Scratch to program sounds or actions that respond to spaces on the board game.

 Project Steps

  1. Plan Out Your Board Game

  2. Program Each Event

  3. Wire Your Board Game

  4. Finish Your Board Game

Plan Out Your Board Game

Your first challenge is to brainstorm what your board game will look like. Think about the board games you've played to help you come up with ideas. 

As you talk with your partner, decide on:

  • A theme (like Escape Room, Journey Around the World, Minecraft)
  • What is the goal? Are players trying to get objects, reach the "End," win the most money, defeat an enemy?
  • How will players reach the goal?
  • What obstacles might stand in the way?

To make this a math game, players will need to solve an equation before making a move; they will add or subtract fractions with unlike denominators. One way to do this is to require players to draw and solve an equation card before moving. You will need to write these equations. (Be sure to have the answers written somewhere so the players could check their work).

Create a sketch of your board game and come up with 2-3 events that will happen when players land on the spaces. Pick 5-7 spaces on the board where these events will happen.

These events can include:

  • Player moves forward X number of spaces
  • Player moves backward X number of spaces
  • Player gets money, tokens, or an object he/she needs to win
  • Player trades spaces with another player
  • Player gets "stuck" and can't move until his/her next turn

Program Each Event

Your next challenge is to develop programs for each event. For each space that has a special event, use Scratch block-coding to program a sound or animation to signal that the event has occurred.

Choose an Event Block for each type of event. When you create your board later, you will connect the spaces to the correct sound or animation using Makey Makey.

For example, when an Escape Room player lands on a clue space, an alarm would sound, and then Scratch could play a sound clip that says, "You have found a clue!" When a player lands on a "false" clue space, an alarm would sound, and then Scratch could play a sound clip that says "No clue here! You lose one minute of search time."

Wire Your Board Game

Your final challenge is to create and wire the board game.

Wire the Path

  1. Sketch the path of the spaces on the board game.
  2. Place copper tape, wire, or tin foil on or underneath the spaces where each event will happen.
  3. Use copper tape, wire, or tin foil to connect the spaces for each type of event. The game piece will be used to complete the circuit and start the Scratch program for each event. How do you keep the events separate if the wire path crosses? Will you use a series or parallel circuit? Test it out on a blank piece of paper.

Test Each Circuit

  1. Connect the alligator leads from the Makey Makey Collection to the board game and the circuit board.
  2. Create the game pieces. What materials can you use for the game pieces? Test out a few materials from the Makey Makey Accessory Collection.
  3. Test your board game. How do you connect multiple game pieces? What happens if multiple players land on the same type of space? Do parallel and series circuits change what happens?

Finish Your Board Game

Now that your programming and circuit is complete, you can add the finishing touches so it is ready to play. Add illustrations, a title, game pieces for each player to use, and anything else needed for your custom game. When the game is finished, explain the rules to your partner or another group and start playing!
