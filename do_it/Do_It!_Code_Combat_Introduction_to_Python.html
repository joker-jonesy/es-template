Do It! Code Combat: Introduction to Python

 The Challenge

Your challenge is to learn how to control a character in a computer game. Your challenge is to use Python Script coding language to program your hero in Code Combat to defeat evil and collect gems. As your hero moves through each of the levels you will interpret division of unit fractions by whole numbers and whole numbers by unit fractions, and then you will compute the quotients.

**The first 21 levels in the Kithgard Dungeon are free to all users, but any levels beyond the first must be purchased. You can move into the Forest and complete free levels in that world. If you would like to access more levels and learn more coding skills, ask your facilitator to purchase a premium class subscription on the CodeCombat website!

Get Into Combat!

Log onto Code Combat and click the Try the Game button to get started.

You will start in the first world called Kithgard Dungeon where you will work through challenges using Python.

When you open the first checkpoint, Dungeons of Kithgard, you will need to choose your hero and pick the Python programming language in the box at the bottom of the screen. You must move through the checkpoints in the correct order as you build on your skills.

In the Career Connection section, you already explored some specific ways that division with fractions and whole numbers might occur in a computer game. Now, as you play Cod eCombat, consider whether these ways apply to your game experience or if there are other ways that division is involved.

Level One

On level 1, when the hero moves to the right one space using the code "hero.moveRight(1)," it takes the hero 10 steps. What unit fraction could you write to represent the fraction of the move represented by each step? How many dots on the gameboard does each move of 1 space cover? Complete the following equation:

1 space ______ (unit fraction) = ________ dots

How could we reverse this concept? Complete the following equation:

____ dots ______ (unit fraction) = ________ space

How would you find the fraction of the move represented by 5 dots? Complete the following equation:

5 dots ______ (unit fraction) = _____ space (this should be a fraction!)

Level Two

On level 2, you will see that the hero walks back and forth on the same path. What fraction of the whole path does each dot on the gameboard represent? How many dots make up the entire path? How many dots make up half of the path? After finding these numbers, complete the equation below.

½ path÷ ______ dots = ________ fraction of whole path for each dot

Level Three

What fractions do you see in level 3? Look at how the hero moves. How can you write a fraction that shows his movement? How many dots make up the path? Create an equation using the format from levels 1 and 2.

Build on Your Skills

Move through the different levels as you practice the different commands. You will start with basic commands and work your way to more complicated code like loops and variables. You are able to move on to the next challenge only when you have successfully completed the current challenge, but you can always go back to practice your skills in an already completed challenge. Sometimes there is more than one way to successfully complete the challenge, so try to write the code in different ways to see if it will work! Get creative!

Build on Your Skills

There are tools inside the checkpoint that can help you as you build on your skills. 

Each time you open a new challenge there is a goal page that pops up and reminds you what you need to do including tips and tricks to solve the challenge.

If you need to change your script language or change something about your hero, click Game Menu.

If you need to change your challenge or return to the map, click Map.

If you forget the script commands your hero needs to complete the challenge, look at the Methods list.

If you need help, click Hints for tips and tricks.
