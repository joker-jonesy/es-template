Do It!

Your challenge is to program a robot lawn mower to go over a whole lawn and to use light effects to show when it is mowing. 

Program Your Lawn Mower

Since you need to develop the software program that tells the lawn mower how to move and communicate, you will use a block-coding app called OzoBlockly to write the code, and Ozobot to simulate the actions and test the code. 

 

Warmup Suggestions

To learn to code with OzoBlockly, complete challenges from each set of the activities below. Do as many as you need to get comfortable coding with loops, light effects, and movement.

Create a Simulated Yard

Ozobot can not “mow” a lawn, but you can use Ozobot to simulate the movement and light effects of the lawn mower.

Preparing for Your Simulation:

  1. Simulate a home with a yard. Create a square or rectangular lawn using green-colored paper.
  2. Simulate areas that should not be mowed, like sidewalks, gardens, driveways, fences, the house, and garage. Use a different color paper to represent each area.

TIP: Make sure all papers are lying flat. You may need to tape them together or glue them down to a larger sheet of paper.

Write the Lawn Mower Code

It is time to write the code. Start with a program that tells Ozobot how to mow the lawn and stay on the grass.

Here are  the technical requirements:

  1. The robot should light up green while it is moving.
  2. The robot should stop when it is done.
  3. The robot should mow the entire lawn without missing spots or go over the same spot too many times.
  4. The robot should only drive on the lawn.

STEPS: 

  1. Go to Ozoblockly.com. Click on the Get Started button.
  2. Write the code. Drag and drop blocks so they click together.
  3. Calibrate Ozobot.
  4. Load the program.
  5. Run the program to test how you did.

Reflections

  • Take a video of Ozobot “mowing” your simulated lawn.
  • Take a screenshot of your code.
  • Record your observations:
    • What did you do to try to complete this project successfully?
    • What challenges did you face?
    • How did you overcome those challenges?
    • Compare your code with another group. How did they meet the challenge? What did they do similarly? What was different about their solution?

Program Your Robot to Mow ANY Lawn

Now change your program so it can figure out how to mow different lawns.

Preparing for Your Advanced Simulation:

  1. Create a lawn in any shape.
  2. Create more sidewalks, gardens, etc. or reuse the ones you had from before to design a new yard.
  3. While Ozobot mows it should light up green.
  4. If Ozobot drives on the sidewalks or driveway, it should light up red to show that it is not mowing.
  5. When you write the code, make sure Ozobot knows not to drive into buildings and things that can break it. Also, make sure it does not drive through and ruin any gardens.
  6. Remember to take video and screenshots, and record your observations.

You can use toolbox 2 to complete this task, but you may want to use toolbox 3 or 4 since they have Logic Statements.
