What is Vectr?

Vectr is a free, online graphics editor that you can use to create vector graphics. A vector graphic is a graphic that you can change the size without changing the quality of the picture.That means you can stretch or shrink your graphic without messing it up. Instead of using pixels, vector graphics use mathematical equations to draw a design.

Vectr can be used to create any 2D graphic. You can use Vectr to create graphics for cards, icons, websites, illustrations, logos, posters, presentations, and much more. Then export your file to print.

How Do You Use Vectr?

Create an Account

Go to Vectr to create an account. If you want to save your work, an account is required.

Create a New File

To start a new file, click Create File in the top left corner of the menu bar. You’ll automatically be brought into the work space.

Set the Page Size

You can select a template size from the dropdown menu or customize your own. In the board (middle of the workspace), the page is outlined by a gray box.

Add Text

Select the Text Tool  in the toolbar (left side of the workspace) to add text to your project.  Click the middle of the board and type.

Adjust Text

A font filter section will appear in the editor. The font filter allows you to adjust the font size, font family, font weight, line height, letter spacing, and boundary alignment.

Add Images

To add an image, click the insert tool. Select an image file from your computer and it will be added to your workspace. Images can also be added to the workspace by dragging an image from your computer and dragging it into the workspace.

Save

Vectr will automatically save projects if you are in your account. To access other saved projects click My Profile with the left arrow in the top left-hand corner.

Export a File

Click the Export Image Button located on top of the toolbar. Rename your file by clicking and typing in the file name section. Then, select Export and Download to save your creation.

Explore On Your Own

Access tutorials and lessons to help you learn and practice the skills needed to design your own logo or web icon.
