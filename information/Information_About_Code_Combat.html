What is Code Combat?

Code Combat is a web-based role playing game (RPG) where you will learn to use Python Script coding language to control a character to defeat evil and collect gems.

How Do You Use Code Combat?

Create an Account/Login

Open Code Combat and click:

  1. I’m a Student Button.
  2. Select Play Now.

You will start in the first world called Kithgard Dungeon where you will work through challenges using Python or Javascript.

When you open the first checkpoint Dungeons of Kithgard, use the "Change Hero or Language" button to choose your hero and pick the Python or Javascript programming language in the box at the bottom of the screen. You must move through the checkpoints in the correct order as you build on your skills.

Save Your Progress

You will need to join with a class code or create an account to save your progress.  Ask your facilitator for help with creating an account.

Account Options:

  • Class Code - Your facilitator can create a class account and share a class code with you.
  • Sign up for a free account using a username & password.

In Game Tools

There are tools inside the checkpoint that can help you as you build on your skills.

  • If you need to change your script language or change something about your hero, click Game Menu.
  • If you need to change your challenge or return to the map, click Map.
  • If you forget the script commands your hero needs to complete the challenge, look at the Methods list.
  • If you need help, click Hints for tips and tricks.

Goal Page

Each time you open a new challenge there is a goal page that pops up and reminds you what you need to do and tips and tricks to solve the challenge.

Explore On Your Own

Use the Student Quick Start Guide to explore the different tools and information shown on the screen during your game.
