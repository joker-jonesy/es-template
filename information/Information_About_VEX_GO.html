What is VEX GO?

VEX GO is a construction system that helps students learn to code and engineer. You can use VEX GO to program something to move by using block code or with batteries. You can also use VEX GO to simply build something. 

 

How Do You Use VEX GO?

What is the Eye Sensor?

The eye sensor connects to the brain of the robot. The eye sensor detects if there are any objects present. The eye sensor can also detect the color of any object (red, green, or blue). 

VEX Go Pieces 

VEX GO has many different pieces you can use to build with. 

Pins

Pins are used to connect different pieces. 

Plates

Plates are wider than pins which allows you to connect many different pieces. 

Beams

Beams have holes to be able to connect other pieces.

Angle Beams

Angle beams allow builds to have structures placed at an angle. 

Large Beams

Large beams have many holes to be able to connect many pieces. 

Connectors

Connectors are used to connect one piece to another.  

Sensors

Sensors let your robot collect information from the world around them. The VEX GO sensors are the eye sensor, LED bumper, and electromagnet. Can you figure out what each one does?  

VEX GO Electromagnet

If you want to incorporate the electromagnet into your build, here is some information about the code blocks. The electromagnet can be used to pick up the colored drop discs in your kit. You can use blocks to:

  • energize, or turn on, the electromagnet
  • set the power percentage
  • set the electromagnet duration, or how long it will be on
