What is PowerPoint?

PowerPoint is a presentation program. With PowerPoint you can:

  • Create presentations from scratch or use a template
  • Add text, images, and videos
  • Add transitions and animation
  • Share your presentation with others

How Do You Use PowerPoint?

Create a Presentation

To create a new presentation:

  1. Open PowerPoint.
  2. Select an option:
  • Select Blank Presentation to create a presentation from scratch.
  • Select one of the templates.
  • Select Take a Tour, and then select Create, to see tips for using PowerPoint.

Add a Slide

To add a slide:

  1. Select the slide you want your new slide to follow.
  2. Select Home > New Slide.
  3. You can choose an existing layout, or, if you want to put text and images wherever you want, you can choose a blank slide. 

Add and Format Text

To add and format text:

  1. Place the cursor where you want, and type.
  2. Select the text, and then select an option on the Home tab: Font, Font size, Bold, Italic, Underline, …
  3. To create bulleted or numbered lists, select the text, and then select Bullets or Numbering.

Save Your Presentation

To save your presentation:

  1. Select File > Save As.
  2. Select your student folder.
  3. Rename your Project (Name).
  4. Click on Save.

Find Your Version

Find your version of PowerPoint:

  1. Open a new PowerPoint Presentation (any theme works).
  2. Click on File.
  3. Click on Accounts. The version will be to the right.
