What is Code.org?

Code.org is a platform (website) that provides an introduction to basic coding and computer science through a collection of tutorials, interactive games, puzzles, and mazes. Code.org also allows you to complete courses and design your own apps and games

How Do You Use Code.org?

Getting Started

You do not need an account to use Code.org; however, if you want to save your progress you will need to create an account using an email. If you want to get started without creating an account, visit the homepage at Code.org and click Learn and choose a course or tutorial.

Blockly Block Programming

Code.org uses a coding language called Blockly.  Each block represents a line of code or command that can be stacked together like puzzle pieces to create a computer program.

Learn Tab

Select the Learn tab at the top of the Code.org website to learn the basics of coding.  Under the Learn tab you will find:

  • Hour of Code Activities, Games, and Mazes
  • Short videos explaining how computers, the internet, and AI (artificial intelligence) work
  • Self-paced courses

Projects Tab

Under the Projects tab, you can click View Project Ideas for starter projects in Game Lab, App Lab, and Web Lab. These include project descriptions, tips, and demo projects you can remix and make your own. 

  • App Lab is a programming environment where you can make simple apps.  Design an app, code with blocks or JavaScript to make it work, then share your app in seconds.
  • Game Lab is a programming environment where you can make simple animations and games with objects and characters that interact with each other.
  • Web Lab is a programming environment where students can make simple web pages using HTML and CSS.  Design your web pages and share your site.

The immersive reader feature reads lesson instructions aloud, but it also provides more functionality by allowing you to make adjustments to lesson text, language, or the structure of sentences.

If you get stuck on a level and need help, click on the hints section of the instructions box.

Explore On Your Own

Get inspired! Browse the Project Gallery and remix!

Go to the Code.org Project Gallery and look at what other students around the world have shared. You can view a project, see the code, and remix it to make it your own!
