What is Google My Maps?

Google My Maps is a map-making app. It is used for making personalized maps. The maps can be customized with pictures, icons, and titles. The maps can be shared and built collaboratively.

Once a map is created, there are tools like a measurement tool that can be used to find the distance between locations and the area. There is also a direction tool that can be used to find directions between locations.

How Do You Use Google My Maps?

Getting Started with Google My Maps

  • Go to Google My Maps.
  • You will need a Google account.
  • Click Get Started.
  • Sign in to your account.
  • Click the red Create a New Map button.
  • Name the Map by clicking Untitled Map.

How to Add the Locations to the Map

  1. In the Search box, type the starting address.
  2. Click the plus sign (+) to add the location to the map.
  3. Type the name of the first national park in the Search box and press enter. (Hint: Include the word “national park” in the Search box. For example, if visiting Yellowstone, type “Yellowstone National Park”.)
  4. Click the plus sign (+) to add the national park to the map.
  5. Repeat steps 3 and 4 until all of the national parks are added to the map.

How To Change the Color and Style of Icons

  1. Hover the cursor over the locations. A bucket will appear.
  2. Click the bucket.
  3. Select a color
  4. Select an icon.
  5. Repeat for each location.

How To Find Directions

  1. Click the first location.
  2. Click the Directions icon.
  3. Type the next location in Box B. Remember to press Enter or Return on the keyboard.

How To Complete Your Map

  1. Click the 3 vertical dots next to the directions. Select Step-by-Step Directions to see a list of directions from the first location to the second location.
  2. Check the completed map.


Explore On Your Own

Create a map for one of your favorite places to visit.

  1. Start by finding your favorite place on the map using the Search box.
  2. Add it to your map.
  3. Change the color of the pin and select an icon to represent that place.
  4. Repeat for other locations in the area (restaurants, hotels, tourist attractions).
  5. View the step-by-step directions.
  6. Check your completed map.
