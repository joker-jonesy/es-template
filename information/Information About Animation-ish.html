What is Animation-ish?

Animation-ish is a web-based animation program that allows you to draw and animate your ideas. Animation-ish  has three levels:

Level 1: Wiggledoodle-ish

You can use Wiggledoodle-ish to learn the basics of animation. You will draw three pictures and play them fast.

Level 2: Flipbook-ish 

Flipbook-ish allows you to draw lots of pictures. More drawings allow you to create a longer, more detailed story.

Level 3:  Advanced-ish 

Once you are comfortable using Wiggledoodle-ish and Flipbook-ish, you can move on to Advanced-ish. With Advanced-ish you can add a background, foreground, and art from the vault.

How Do You Use Animation-ish?

Start Wiggledoodle-ing

STEPS

  1. Start the Animation-ish software. If you don't have the program installed on your computer, go to the web version and use your username and class code provided by your facilitator to sign into Animation-ish.
  2. Go to FableVision Learning to watch the lessons.
  3. Watch Lesson 1.
  4. Go back to the Animation-ish application and select Wiggledoodle-ish. The workspace will open.
  5. Try out what you learned from the Lesson 1 video.

Start Flipbook-ing

STEPS

  1. Start the Animation-ish software. If you don't have the program installed on your computer, go to the web version and use your username and class code provided by your facilitator  to sign into Animation-ish.
  2. Go to FableVision Learning to watch the lessons.
  3. Watch Lesson 2 to learn how to make frames in FlipBook-ish.
  4. Go back to the Animation-ish application and select FlipBook-ish and the workspace will open.
  5. Try out what you learned from the Lesson 2 video.

Start Advanced-ish

Advanced-ish is the most advanced level of Animation-ish. Advanced-ish has more tools and resources than Wiggledoodle-ish and FlipBook-ish.

STEPS

  1. Start the Animation-ish software. If you don't have the program installed on your computer, go to the web version and use your username and class code provided by your facilitator to sign into Animation-ish.
  2. Go to FableVision Learning to watch the lessons.
  3. Watch Lessons 3, 4, and 5.
    Lesson 3: Advanced-ish (Part 1)
    Lesson 4: Advanced-ish (Part 2)
    Lesson 5: Advanced-ish (Part 3)
  4. Go back to the Animation-ish application and select Advanced-ish. The workspace will open.
  5. Use what you learned in the lessons to make your own animation with Advanced-ish.

Animation-ish Tools

Drawing Tools

Use the drawing tools to:

  • Select items - Arrow
  • Paint  - Brush
  • Fill a space with color - Paint Can
  • Erase

Brush Size

Use the brush size tool to select the size of your paint brush.

Add/Change Colors

Click on the color palette to select or change the color.

For more color options:

  1. Double click on any color and a character with wings will appear.  You can click on any color you see on his wings, face or body.  The color you have selected will show up in the thought bubble above his head.  
  2. Click OK to start using the color you have selected.

Undo or Delete

Use these tools located on the top toolbar to undo an action or delete an object.

Control Your Animation

Use these tools to scroll through pages, adjust speed, loop or play your animation.

Save, Share, and Export Your Animation

To save your animation:

  1. Click on the Save Button
  2. Select your student folder.
  3. Title your animation in the File Name box.
  4. Click save.

To share your animation:

  1. Click on the Share Button.
  2. Select the type of file you want for your animation. If you are not sure what to choose, ask your Facilitator.

To export your animation: 

  1. Select your student folder.
  2. Title your animation in the File Name box.
  3. Click save.
