Information About Dash

What is Dash?

Dash is a robot that can be programmed to move, avoid obstacles, make noise, shine light, or even interact with another robot.

How Do You Use Dash?

Parts of Dash

Blockly Coding App

Dash can be programmed using the Blockly coding app. The Blockly coding app lets you drive and control your robot using written commands. In this app, you will use code blocks to build a program.

When you drag and drop blocks together you create a set of instructions for your robot to follow.

Sequences

Dash will read the set of blocks in order, or sequence, from top to bottom. A sequence is an ordered set of instructions for the robot to perform. Order matters!

Each time your program Dot or Dash, you will start with a When block. The When block tells the robot what starts the program. Everything you want your robot to do is attached to the When block, in order.

Moving Dash

Dash can move forward and backward with DRIVE blocks. Dash moves in jumps, or increments, of 10 centimeters (cm). Dash can also turn left and right at any angle.

Code Blocks

There are many types of code blocks you can use in your program. Click on the block types below to learn more.

Select each tab to learn more.

START blocks control how your program will start. Use a block beginning with “When” to pick the event that will start your program. What events might start your program? Think about the sensors your robot has and how they can be used to detect events.

DRIVE blocks control where and how Dash moves. Forward and backward blocks use centimeters as the units to measure how far it goes. To make Dash turn, you will use degrees as the units to measure how far it should turn.

LOOK blocks make Dash’s head move any way you want. The LOOK blocks use degrees as units of measurement. You can change the number of degrees to turn Dash’s head to the left or right.

LIGHT blocks control Dash’s lights to give your program some color.

SOUND blocks let Dash play sounds. Use the drop down menus to add sound effects to your programs.

Saving Projects

In the Blockly app, you can create and save your own programs. You can also load sample projects if you want to see an example program.

Tap the hamburger menu on the top left hand side of the app to create a new project. You can select a preset or start off from scratch with a blank project!

If you start a new project or open an old project and make changes, your changes will automatically save. You can find all your projects in the menu under My Projects. To change your project name or icon, tap on the icon next to the hamburger menu!

Explore On Your Own

Practice Programming with the Blockly App

The puzzles in the Blockly app will teach you how to program Dash with code blocks.

  1. Open the Blockly app, and press the "play" button.
  2. Select your profile.
  3. Click on Puzzles.
  4. Click on the first puzzle and follow the directions. When you complete it, more puzzles will unlock.
