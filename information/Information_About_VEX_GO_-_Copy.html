What is VEX GO?

VEX GO is a construction system that helps you learn to code and engineer.

Read the VEX GO Storybook to learn more about this tool.

You can use VEX GO to build, but you can also program VEX GO to move using block code.

How Do You Use VEX GO?

VEX GO has many pieces you can use to build with.

Select each item to learn more.

Pins

Pins are used to connect different pieces.

Plates

Plates are wider which allow you to connect many other pieces to it.

Beams

Beams have holes to be able to connect other pieces.

Large Beams

Large Beams have more holes to be able to connect other pieces.

Angle Beams

Angle Beams allow builds to have structures placed at an angle.

Connectors

Connectors are used to connect one piece to another.

Sensors

Sensors let your robot collect information from around them. The VEX GO sensors are the Eye sensor, LED Bumper and Electromagnet. Can you figure out what each one does?

What is VEXcode GO?

VEXcode GO is the website or app where you will use block coding to program your animal.

You will use blocks that snap together to program your animal.

The program workspace where you will drag your blocks to create a program for your robot.
