Do It! Ozobot: Mow a Lawn

 The Challenge

Your challenge is to program a robot lawn mower to go over a whole lawn and to use light effects to show when it is mowing. Then, count how many passes the robot takes to mow the lawn to identify the fractional value of each pass.

 Project Steps

  1. Create a Simulated Yard

  2. Write the Lawn Mower Code

  3. Program Your Robot to Mow the Lawn a Different Way

Create a Simulated Yard

Ozobot cannot “mow” a lawn, but you can use Ozobot to simulate the movement and light effects of a lawn mower. 

Preparing for Your Simulation:

  1. Simulate a home with a yard. Create a rectangular lawn using green-colored paper.
  2. Simulate areas that should not be mowed, like sidewalks, gardens, driveways, fences, the house, and garage. Use a different color paper to represent each area.

Make sure all papers are lying flat. You may need to tape them together or glue them down to a larger sheet of paper.

Write the Lawn Mower Code

It is time to write the code. Start with a program that tells Ozobot how to mow the lawn and stay on the grass.

Technical Requirements:

  1. The robot should light up green while it is moving.

  2. The robot should stop when it is done.

  3. The robot should mow side to side across the lawn.

  4. The robot should mow the entire lawn without missing spots or going over the same spot more than once.

  5. The robot should only drive on the lawn.

STEPS: (For a review of these steps, watch the Ozoblockly Tutorial again.)


Direct Link: Ozoblockly Tutorial

  1. Go to Ozoblockly.com. Click on the Get Started button.
  2. Write the code. Drag and drop blocks so they click together.
  3. Calibrate Ozobot.
  4. Determine where your robot will start on the paper.
  5. Write the program.
  6. Load the program.
  7. Run the program to test how you did.
  8. Take a video of Ozobot “mowing” your simulated lawn.
  9. Draw lines along the lawn to show the partitions made with each pass. Write the fraction of each pass as 1 over the number of partitions, or 1/number.
  10. Take a screenshot of your code.
  11. Add labels to your screenshot to show the fractions.

Program Your Robot to Mow the Lawn a Different Way

Program your robot lawnmower to mow the same lawn but to mow going the opposite direction  by moving up and down instead of side to side. Follow the same steps above, then analyze how changing the direction changes the fraction pieces of the lawn. 

  1. Now, rewrite the code so that Ozobot mows the same lawn by moving up and down instead of side to side. This will make a different fraction.
  2. Count the partitions. Write the fraction.
  3. Reflect: How do the two fractions compare? How does this change affect Ozobot? How does this change affect the time Ozobot needs to mow the lawn?
  4. Make a second rectangular lawn to mow.
  5. Program Ozobot to mow back and forth.
  6. Count the partitions. Write the fraction.
  7. Next, rewrite the code so that Ozobot mows up and down.
  8. Count the partitions. Write the fraction.
  9. Reflect: How do the two fractions compare? How does this change affect Ozobot? How does this change affect the time Ozobot needs to mow the lawn?
