Do It!

Your challenge is to create a computer game with Kodu and then add features to make it better. First, you will create the Shooting Fish game that programs a fish to shoot missiles. Next, you will improve the Shooting Fish game and make it even better.

Create Your Shooting Fish Game

How to Save Your Work

STEPS:

  1. Click the Configure Kodu icon. This will open the Kodu Settings menu.  
  2. Next, choose the location where you will save your work. Click the button under Save Folder and browse to your student folder. 
  3. Click OK.

Important: Do this EVERY day before you start using Kodu.

Launch Kodu Game Lab

STEPS:

  1. Click the Kodu Game Lab icon.
  2. Click the Load World menu option.
  3. Find the Shooting Fish world and practice the steps you learned in the First Game Tutorial video.

Go to the What You Should Know: First Game Tutorial section to review the tutorial.

First Game

STEPS:

  1. Load Shooting Fish world
  2. Add Kodu object
  3. When: spacebar pressed - Do: shoot missile
  4. When: left arrow key is pressed - Do: turn left
  5. When: right arrow key is pressed - Do: turn right
  6. Save your game

Improve Your Game

Were you able to program Kodu to shoot fish? 

If so, you have created your first computer game. Can you think of ways to make the game better?

  • You can watch more video tutorials and then apply what you have learned to make changes to the Shooting Fish game. Go to the Kodu website and choose a video tutorial to watch.
  • You can explore Kodu by clicking on different objects and When - Do options to see what happens.
    Tip: If you hover your mouse cursor over a menu item, a description of that item will appear.

There are lots of ways to make your Shooting Fish game better.

Ideas to Improve Your Game

  • Program the game to keep score.
  • Program the game to show a Winner screen when you score a certain amount of points.
  • Program different point scores for hitting different objects or colors of fish.
  • Add a timer so the game ends after a certain amount of time.
  • Add obstacles for Kodu to shoot around.
  • Add other objects for Kodu to shoot.
  • Program Kodu for new behaviors, like moving around.
  • Program the fish to move differently.
  • Change the terrain.
  • Add new characters to your world to interact with Kodu.
