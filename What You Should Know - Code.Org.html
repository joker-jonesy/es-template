Important Vocabulary

  • Computer Code - a set of instructions that tells an electronic device what to do
  • Command - a single line of code telling an electronic to do one thing

Computer Code is Everywhere

Computer code is a big part of our everyday lives. Computers, tablets, smartphones and even electronic toys and games, microwaves, cars, and cameras all need computer code to function.

What is Computer Code?

We are surrounded by computer code. But what is it?

Computer code is a set of instructions that tells an electronic device what to do. Each line of code is a single instruction or command.

While computers and electronic devices seem smart, they only appear this way because a human (e.g. computer programmer, software engineer, developer) wrote a set of instructions for them.

The computer code tells your computer what to do when you interact with it, such as clicking a mouse or typing on the keyboard. It also controls what a video game character does when you press a button. 

In fact, every app (location) and computer program you use is created using many lines of code that tell the device how to act and what to show on the screen.
