The Challenge - Scratch: Program to Solve Word Problems

The Challenge

Your challenge is to use Scratch to program a computer animation that teaches math. You can add to an existing program or plan and create your own animation "from scratch." You could have one character present a multiplication or division word problem and the other could solve it. Or, your characters could act out a word problem. Or perhaps you have another idea of how you could make an animation about math. 

You will use Operator blocks in your script in order to include multiplication or division in your code.  The script you create will help you practice writing and solving word problems, and find unknown whole numbers in multiplication and division equations.

What I'll Learn 

  • I CAN design an animation on Scratch that uses Operator blocks in the script.
  • I CAN create a code that incorporates multiplication or division.

Stuff I'll Need

  • Scratch Website
  • Scratch Software
    • Mac: in your dock
    • PC: All Applications folder
