Interactive Programs

Making a program interactive means that it does different things based on what you do, and the instructions you give it.

For example, your alarm clock goes off at the same time every morning because you program it to do that. Sometimes you may want it to behave differently. If you hit the "snooze" button to sleep in, the alarm sounds at a later time, and yet you do not have to reset the alarm.

If you want a character in a game to move, you click a mouse or push a button. You do not have to rewrite the program every time.

Learn From Other Projects

Each Scratch project includes ideas for making changes. If you want to see the code for an animation or game, click the See inside button.

You can change the code to make your own version of the project. Changing the code helps you learn different ways to code. And, it helps you find new project ideas.
