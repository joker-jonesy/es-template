Do It!

In this Liftoff, you will learn how to control a character in a computer game. Your challenge is to use Python Script coding language to program your hero in Code Combat to defeat evil and collect gems. 

Getting Started

Log onto Code Combat and click Try the Game button to get started. 

You will start in the first world called Kithgard Dungeon where you will work through challenges using Python.

When you open the first checkpoint Dungeons of Kithgard, you will need to choose your hero and pick the Python programming language in the box at the bottom of the screen. You must move through the checkpoints in the correct order as you build on your skills. 

Build on Your Skills 

Move through the different levels as you practice the different commands. You will start with basic commands and work your way to more complicated code like loops and variables. You are only able to move on to the next challenge after you have successfully completed the challenge, but you can always go back to practice your skills in an already completed challenge. Sometimes there is more than one way to successfully complete the challenge, so try to write the code in different ways to see if it will work! Get creative!

Helpful Tools

There are tools inside the checkpoint that can help you as you build on your skills.

Each time you open a new challenge there is a goal page that pops up and reminds you what you need to do and tips and tricks to solve the challenge. 

  • If you need to change your script language or change something about your hero, click Game Menu.
  • If you need to change your challenge or return to the map, click Map.
  • If you forget the script commands your hero needs to complete the challenge, look at the Methods list. 
  • If you need help, click Hints for tips and tricks.
