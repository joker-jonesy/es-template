Do It!

Your challenge is to program Lego WeDo sensors to control the behavior of Scratch Sprites.

With your programmed sensors, you could move Scratch Sprites through a maze, up and down on a see-saw, and more.

STEPS:

  1. You may need to load the WeDo blocks to Scratch program before you can start to program. Follow the Software Check in the What You Should Know section.  
  2. Program your Scratch Sprites using both the motion and the tilt sensor.
  3. Once you have completed the step-by-step programming instructions for both sensors, you will be on your way to creating your own program. 

If you need more information about using WeDo with Scratch, go to the What You Should Know section. 

Load WeDo Blocks

To program WeDo with Scratch you need special blocks to control the motor and sensors.

  1. Open Scratch.
  2. Select, Create.
  3. Click on More Blocks.

If you do not yet see the Lego WeDo blocks, follow these steps:

  1. Click the Add an Extension button.
  2. Click WeDo 1.0.

 

Still no blocks? Go back to What You Should Know: Software Check. 

Program Your Motion Sensor

Start by making sure the motion sensor works.

STEPS:

  1. Connect the USB hub into the computer.
  2. Connect the Motion Sensor to the USB hub.
  3. Open Scratch.
  4. Write a simple Scratch program to test the distance sensor. Make the “Meow” sound play when the motion sensor detects movement.
  5. Move your hand in front of the sensor. You should hear a “meow.”

Program Your Tilt Sensor

Explore the tilt sensor block in Scratch.

STEPS:

  1. Connect the USB hub to the computer.
  2. Connect the Tilt Sensor to the USB hub.
  3. Open Scratch.

The tilt sensor is more complex. You will need to explore it in a little more detail. 

Try this:

  1. Start a new Scratch program.
  2. Combine these blocks:
    • When tilt = 1, move 10 steps.
  3. Now move the tilt sensor.

Which position makes the cat on the screen move?

Tilt Sensor Tests

Here are several tilt sensor tests. Try each one before you move on.

  1. Combine these blocks. With your previous blocks (when tilt = 1, move 10 steps) you should have five different sets of blocks. 
    • When tilt = 2, make meow sound.
    • When tilt = 3, change color by 25.
    • When tilt = 4, say Hello!
    • When tilt = 0, go to x: 0, y:0.
  2. Move the tilt sensor.
  3. Observe the behavior of the cat on the screen.
  4. Repeat the steps above but change the numbers and the behaviors.

Were you able to control the behavior of the cat with the tilt sensor? What is the meaning of each tilt number?

Write Your Own Program

Use what you learned in the above tasks to create your own program.

Create an animation in Scratch that you control using one or more Lego WeDo sensors.

Here are some ideas to get you started:

Recreate a popular computer or video game.

Write a program that uses one of the WeDo sensors to control a Sprite in the game.

Build an alarm.

Program the motion sensor to sound an alarm when someone tries to move an object.

Complete a maze.

Program the tilt sensor to move a Sprite through a maze. Have a friend try to complete the maze.

Use the tilt sensor.

Build a see-saw with Lego. Place the tilt sensor on the see-saw.

Program Scratch Sprites to move up and down on the computer screen when the
see-saw tilts.

Use the distance sensor.

Use the distance sensor to keep a car from crashing into a wall.

Make a Lego car with a built-in motion sensor. Associate your Lego car with a car sprite in Scratch. Keep your sprite and Lego car from crashing into walls.
