Suggestions for Use

The rotation map templates (linked below) provide a tool for planning and presenting the technologies that each class or grade level will use during their time in the SmartLab HQ. Each map can be edited to reflect the grade level or class and the specific set of technologies for that grade level or class.​

  • The Individual Project Slide templates can be used to provide a more detailed view of a specific Project Starter that you want your class to complete. This slide can be a valuable way to introduce a whole-class project and the specific challenge and learning objectives for that project.​
  • The Technology Graphics slides serve as a library of images to use when planning the rotations for your SmartLab classes. The graphics are organized alphabetically by technology name and match the graphics students will see on the Project Starters in LearningHub.​

We recommend copying and pasting the appropriate technology graphics into the desired template for each class period and projecting the modified rotation map to help students know which projects they should be working on and which projects they can anticipate next in their rotations.
