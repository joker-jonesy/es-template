What is Scratch?

Scratch is a web application that allows you to:

  • Write simple computer programs.
  • Create computer stories, games, and animations.
  • Share stories, games, and animations with others.

How Do You Use Scratch?

Getting Started

The Create version of Scratch is free and does not require a login; however, you cannot save your work without an account. To create an account in Scratch, you will need an email address, birthday, username, and password. With an account you can:

  • Save your work in Scratch
  • Share your Scratch projects online
  • Manage your Scratch projects

Project Editor

When you create in Scratch, you will work in a space called the project editor window. The project editor window has all the resources and tools necessary to create and run your own program.

Stage

The stage in the project editor window shows your current characters and objects. It is also where you can see your program in action as you edit it. What you see on the stage is how your program will actually look to your audience and behave when you are finished.

Sprite List

Scratch calls persons, places, and things sprites. All sprites used in your program show up in the sprite list. Scratch comes with a collection of pre-made sprites to use, but you can also upload or create your own if you cannot find exactly what you are looking for.

To find a new sprite, click on the character icon above the sprite list. This opens a Sprite Library.

Simply choose a new sprite and it will be added to your sprite list and appear on the stage. To upload a sprite image, click the folder icon. To create your own, click the paintbrush icon.

Code Blocks

Scratch uses blocks instead of symbols, numbers, and letters for coding computer programs.

Each block represents a line of code. These blocks of code snap together like puzzle pieces.

By connecting blocks of code, you create a computer program.

There are different kinds of blocks. Each type comes in a different color.

When you use code blocks to program you may have to “debug” your program. If you run your program and it does not work like you thought it would, you have to fix or “debug” it. Check each block. Check for the right colors, numbers, and words. Make sure each block is connected to another block. The more you practice debugging, the better and faster you will get at fixing problems.

Scripts Area

In order to control sprites, you must drag the blocks into the scripts area of the workspace (the middle). Only the blocks in the scripts area control the sprites. This is where you create your program by adding code blocks together. Click the green flag to run your program or script, and then watch the stage to see how the sprite behaves.

Note: Every sprite you have in your program has its own script code. To switch between each sprite's script area, just click the "Scripts" tab above the blocks section and then select the sprite from the sprite list that you want to build a script for.

Green Math Operator Blocks

For some activities you will need to use a specific block in your program. Green Operator blocks are an example of a special block. Green Operator blocks in Scratch are used to add math to a script. When you type numbers into the spaces in green Operator blocks, they solve math problems. For example, you can use the multiplication and division blocks to create scripts that multiply or divide numbers.

Explore On Your Own

Learn From Other Projects

One of the best ways to learn how to code something in Scratch is to experiment: Find a project that you like, look at the code, and then try making changes to it. If you want to see the code for an animation or game that has already been created, click the See inside button.

You can change the code to make your own version of the project. Changing the code helps you learn different ways to code. And, it helps you come up with new project ideas.
