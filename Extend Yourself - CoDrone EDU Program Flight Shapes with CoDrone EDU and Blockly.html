Extend Yourself - CoDrone EDU: Program Flight Shapes with CoDrone EDU and Blockly

Special Blockly Challenges

Complete the special Challenge at the end of Blockly Lessons 1.2, 1.3 and/or 1.4. These are located in the final step before the Lesson Complete step.

Lesson 1.2 Challenge: CoDrone EDU Pushups
See if you can apply what you learned to this programming challenge: Can you program your drone to take off and land 3 times in a row?

Lesson 1.3 Challenge: Rewind
This challenge includes code. Run the code, then write a new program that runs the whole flight path in reverse.

Lesson 1.3 Challenge: Shapeshifter
Write a Blockly program using the **Turn** block to fly in different equilateral polygons. This includes needing to figure out the correct angles.

Lesson 1.4 Challenge: More Variables!
Explore what else can be done with variables. What flight patterns can you create if you add variables for duration, power, and degrees?

Vertical Flight

Instead of having CoDrone fly horizontally (parallel with the floor) to make your shapes, program it to fly vertically (up and down, perpendicular with the floor).

Draw and Program

Draw any closed shape and program CoDrone to fly in that shape. It doesn’t even have to be a quadrilateral or a named shape! They key is that the flight pattern matches your drawing.
