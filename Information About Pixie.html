What is Pixie?

Pixie is a graphics program that can be used to make art, stories, presentations, videos and more. Pixie 4 is the most current version available. Use Pixie to:

  • Draw pictures
  • Tell a story
  • Create a video

How Do You Use Pixie?

Start a New Project

Open Pixie to start a new project.

Quick Start Tools

Add Text

  1. Click on the Text Box Tool.
  2. Double click inside the box to add text.
  3. Use the Options menu to edit your text.

Paint Tool

Click on the Paint button in the top, right corner. The paint tools will allow you to add paint or shapes to your project . When you select a specific tool, the Tool Options will appear.

Library

To access saved pictures and backgrounds, click on the Library Button in the top, right corner.

The tools in the Library will allow you to:

  • Add a Background
  • Add a Sticker or Image
  • Take a Picture Using your Webcam
  • Use an Existing File on Your

Step 2

Save Your Project

To save your project:

  1. Click on Save.
  2. Click on To Computer.
  3. Navigate to your student folder.
  4. Name your Project in the File Name box.
  5. Click Save.

Export Your Project

To export your project:

  1. Select Project.
  2. Select Export.
  3. Under Export choose to Export as a:
    1. Video
    2. PDF
    3. HTML
    4. Image
  4. Click the Computer button and save your work to your student folder.
