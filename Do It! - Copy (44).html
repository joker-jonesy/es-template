Do It!

Your challenge is to use Java Script coding language to program your hero in Code Combat to defeat evil and collect gems.

Getting Started

Go to the Code Combat website. click Try The Game.

When you open the first checkpoint Dungeons of Kithgard you will need to choose your hero and pick the Java Script programming language. If you have to reload the page you must double check that you have Java Script selected as the programming language.

You can double check the correct programming language is selected at any time by looking at the right hand corner of your game screen. 

Build on Your Skills

Move through the different levels as you practice the different commands. You will start with basic commands and work your way to more complicated code like loops and variables. You are only able to move on to the next challenge when you have successfully completed the current challenge, but you can always go back to practice your skills in an already completed challenge. Sometimes there is more than one way to successfully complete the challenge, so try to write the code in different ways to see if it will work! Get creative!

Helpful Tools 

Tools inside the checkpoint that can help you to write the Python program to control your hero include:

Each time you open a new challenge there is a goal page that pops up and reminds you what you need to do and tips and tricks to solve the challenge. 

  • If you need to change your script language or change something about your hero, click Game Menu.
  • If you need to change your challenge or return to the map, click Map.
  • If you forget the script commands your hero needs to complete the challenge, look at the Methods list. 
  • If you need help, click Hints for tips and tricks.
