Information About Vernier Elementary Energy Efficiency 

What is Vernier Elementary Energy Efficiency?

The Vernier Elementary Energy Efficiency kit has four sensors inside. Sensors are used to collect information from the world around you. This kit has:

  • Vernier Color and Light Sensor -this sensor measures color and light intensity (strength). 
  • Vernier Surface Temperature Sensor- this sensor measures the temperature of surfaces.
  • Vernier Temperature Probe- this sensor measures the temperature of liquids, objects, and the air.
  • Infrared Thermometer- this sensor measures the temperature of an object without having to touch it.

    Vernier Light and Color Sensor

    The Vernier Light and Color Sensor collects data, or information, about light and color. It has five settings, called sensor channels:

    • Light
    • UV
    • 615 nm (color sensor - red)
    • 525 nm (color sensor - green)
    • 465 nm (color sensor - blue)

    The light sensor channel collects data about the amount of light that is reflecting off an object or surface. When light reflects off a surface, it hits the surface and then bounces off in another direction.

    The other sensor channels collect data about UV and color. UV is a type of light energy that comes from the sun. You can’t see UV light. The color sensor channels collect data about the color of an object.

    To collect data, you will use the Vernier Graphical Analysis app. The app is used to collect, analyze, or look closely at, and share data collected from the sensors. The Vernier sensors can be connected to the app with a USB cable or Bluetooth. It can be used on an iPad or a computer. 

    How Do You Use Vernier Elementary Energy Efficiency?

    Using Vernier Graphical Analysis

    Click the Vernier Graphical Analysis icon to open the app. Watch this tutorial to learn how to:

    • Connect the sensor
    • Start a new experiment
    • Collect data
    • View data
    • Export (or download and share) data

    Connecting the Sensors via USB Cable

    If your sensor’s battery is low, you can connect it to your computer with a USB cable.

    Explore On Your Own

    To become familiar with the kit, start by connecting the Vernier Go Direct Light and Color sensor following the instructions in the tutorial above. Make sure the Sensor Channel is set to Light.
