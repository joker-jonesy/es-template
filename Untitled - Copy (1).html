The Challenge - UKIT Beginner: Ace in the Hole

The Challenge

Your challenge is to build and code a Golf Club Robot using the UKIT Beginner kit. You will practice your measuring skills using a protractor and determine what angle the golf club needs to be pulled back to achieve the greatest distance of a ball.

What I'll Learn 

  • I CAN accurately measure different angles using a protractor.
  • I CAN sketch different angles when given a specified measurement.
  • I CAN build a robotic golf club using the UKIT Beginner kit.
  • I CAN use programming language to code a robotic golf club to swing and hit a ping pong ball at a given angle.
  • I CAN write a scientifically accurate Claims-Evidence Reasoning paragraph that includes data to support the claim.

Stuff I'll Need

  • UKIT Beginner Kit 
  • Computer or Tablet  
  • UKIT EDU App (downloaded)
  • Protractor
  • Ping pong ball or other ball
  • Pencil and Paper
  • Meter Stick or Ruler
