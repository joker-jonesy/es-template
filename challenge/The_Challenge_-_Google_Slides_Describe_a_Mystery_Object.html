The Challenge - Google Slides: Describe a Mystery Object

The Challenge

Your challenge is to create a Google Slides presentation that describes a mystery object of your choosing. You will give clues using your five senses to help people try to determine what the mystery object is before the big reveal. 

What I'll Learn 

  • I CAN use my five senses to describe an object.
  • I CAN use descriptive language to describe an object.
  • I CAN use collected data to create a line plot.
  • I CAN solve “how many more/less” questions using graphed collected data.

Stuff I'll Need

  • Google Slides
  • Brainstorm Sheet
