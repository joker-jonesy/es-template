The Challenge - Sphero: Algorithms

The Challenge

Your challenge is to learn how to control a Sphero robot so that it can do tricks and follow a path you create. Then, write and test an algorithm based on Sphero’s actions so that you can make the robot repeat itself. Finally, write and solve a word problem based on your algorithm.

What I'll Learn 

  • I CAN write an algorithm that directs Sphero to follow a path and do tricks.
  • I CAN describe the steps of my path with a number sentence.
  • I CAN write and solve two-step word problems.

Stuff I'll Need

  • Graph or Grid Paper
  • Sphero SPRK+ Kit
  • iPad with Sphero EDU App
