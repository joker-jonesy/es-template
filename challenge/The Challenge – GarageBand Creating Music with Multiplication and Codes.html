The Challenge – GarageBand: Creating Music with Multiplication and Codes

The Challenge

Your challenge is to solve multiplication problems and use a code to translate the answers into notes. Then, you will use those notes in GarageBand to create a melody with the piano keyboard. 

What I'll Learn 

  • I CAN solve multiplication problems up to 12 x 12. 
  • I CAN use GarageBand to create and record a musical composition based on my multiplication knowledge. 

Stuff I'll Need

  • GarageBand  
  • Multiplication Problems Worksheet 
  • Code Translation Worksheet
