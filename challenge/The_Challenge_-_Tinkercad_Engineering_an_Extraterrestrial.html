The Challenge - Tinkercad: Engineering an Extraterrestrial

The Challenge

Your challenge is to use Tinkercad to design your own 3D extraterrestrial character. Then, write a description about it.

What I'll Learn 

  • I CAN describe the attributes of shapes.
  • I CAN categorize shapes with shared attributes into groups.
  • I CAN use Tinkercad to design a 3D object.
  • I CAN write a description about my extraterrestrial character.

Stuff I'll Need

  • Paper and Pencil
  • Tinkercad
