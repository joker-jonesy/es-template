The Challenge - VEXcode VR: Shape Art

The Challenge

Your challenge is to program your VEX robot to create a tessellation. 

Robots need step-by-step instructions to do a job. They cannot do anything on their own; they can only do what humans program them to do.

You will first explore why some shapes fit together while others do not. Then you will practice how to program the VEX robot to create shapes. Finally, you will put all these skills together to create tessellations. This will get you ready to think about why some shapes fit together and others do not.

What I'll Learn 

  • I CAN program a VEX robot to create a tessellation.
  • I CAN make an array to show multiplication.
  • I CAN write more than one multiplication sentence to describe an array.
  • I CAN use one shape to create a repeating pattern.
