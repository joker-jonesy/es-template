The Challenge - Code Combat: Introduction to Python

The Challenge

Learn how to control a character in a computer game.

Your challenge is to use Python Script coding language to program your hero in Code Combat to defeat evil and collect gems. As your hero moves through each of the levels, you will interpret division of unit fractions by whole numbers and whole numbers by unit fractions, and then you will compute the quotients. 

**The first 21 levels in the Kithgard Dungeon are free to all users, but any levels beyond the first must be purchased. You can move into the Forest and complete free levels in that world. If you would like to access more levels and learn more coding skills, ask your facilitator to purchase a premium class subscription on the Code Combat website.

What I'll Learn 

  • I CAN use Python script to control characters in Code Combat.
  • I CAN find the quotient of a whole number divided by a unit fraction.
  • I CAN find the quotient of a unit fraction divided by a whole number.
