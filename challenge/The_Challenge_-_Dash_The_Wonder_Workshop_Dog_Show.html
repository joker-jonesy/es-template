The Challenge - Dash: Create an Obstacle Course

The Challenge

Your challenge is to program Dash to perform pet tricks to get it ready for the Wonder Workshop Dog Show! Then, create an obstacle course for Dash to get through, and write numerical expressions to represent each of the commands required to make Dash move.  

What I'll Learn 

  • I CAN program a robot to accomplish different tasks using block code. 
  • I CAN use the “When” code block to program a robot to act in response to an event. 
  • I CAN write expressions using mathematical symbols and the order of operations. 
  • I CAN explain what an expression represents. 

Stuff I'll Need

  • Dash Robot
  • iPad, Computer, or Smartphone  
  • Blockly App
