The Challenge - Animation-ish: Animate Word Problems 

The Challenge

Your challenge is to use Animation-ish (Flipbook-ish) to solve and animate two-step word problems. 

What I'll Learn 

  • I CAN solve two-step word problems.
  • I CAN create an animation to show the solution to the word problem.

Stuff I'll Need

  • Animation-ish
    • If you have Animation-ish installed on your computer, find it in your Dock (Mac) or All Applications folder (PC).
    • If you don't have the program installed on your computer, go to the web version of Animation-ish login (new link).
    • Animation-ish class code for login (provided by your facilitator).
