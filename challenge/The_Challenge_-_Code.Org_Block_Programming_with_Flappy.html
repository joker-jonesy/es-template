The Challenge - Code.org: Design a Math Game

The Challenge

Your challenge is to learn to program using block code, then program a Flappy computer game to help your friends practice their multiplication facts. 

What I'll Learn 

  • I CAN use block coding to create a game.
  • I CAN use what I know about multiplication facts and properties to program a game that helps my friends practice their multiplication facts.

Stuff I'll Need

  • Code.org's Play Lab Website
  • Code.org's Flappy Bird Website
