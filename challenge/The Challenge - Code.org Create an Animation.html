The Challenge - Code.org: Create an Animation 

The Challenge

Learn the basics of how to use GameLab and then use your new skills to create a bouncing ball animation.

What I'll Learn

  • I CAN create variables.
  • I CAN create and animate a sprite.
  • I CAN debug a program.

Stuff I'll Need

  • Code.org GameLab 
  • Section Code (provided by teacher)
  • Picture Password (provided by teacher)
