The Challenge - Kodu: Terrain Area

The Challenge

Discover how to program as you create your first computer game with Kodu. Your challenge is to create terrain in Kodu and use multiplication and tiling to find its area. 

When you make a computer game, you are writing a computer program.

Your program tells the computer how your game is played.

What will you program Kodu to do?

What I'll Learn 

  • I CAN use Kodu to write code.
  • I CAN use Kodu to create terrain.
  • I CAN describe how to find the area of a two-dimensional polygon.
  • I CAN count unit squares to determine the area of a two-dimensional polygon.
  • I CAN use multiplication to find the area of a two-dimensional polygon.

Stuff I'll Need

  • Kodu Game Lab
    • Mac: In your Dock
    • PC: All Applications folder
  • XBOX Controller (optional)
