The Challenge - VEX GO: A Day in My Ecosystem

The Challenge

Your challenge is to build or create an ecosystem for the animal of your choice. Once your ecosystem is created, you will then build your animal using VEX GO Code Base. Program your animal through its ecosystem using block coding.

What I'll Learn 

  • I CAN distinguish between different types of ecosystems and the animals that live there.
  • I CAN program my robot using sequences, events, loops, and variables.
  • I CAN build an animal using VEX GO.
  • I CAN recreate an ecosystem using my desired materials.

Stuff I'll Need

  • VEX GO Kit
  • Web-based Programming: VEXcode GO 
  • VEX Classroom (requires iPad to update firmware)
  • Building/Drawing Materials
  • A Video Recorder (iPad, GoPro, built-in computer camera)
