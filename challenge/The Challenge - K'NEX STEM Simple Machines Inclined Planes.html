The Challenge - K'NEX STEM Simple Machines: Build and Measure Inclined Planes

The Challenge

Your challenge is to build an inclined plane machine so you can show how inclined planes make work easier. You will use a protractor to measure the angles of your inclined planes. Then, you will compare the attributes of triangles to identify attributes of inclined planes. 

When you think of an inclined plane, you might think of a ramp. However, even this jar has a type of inclined plane in its design.

How is a jar an inclined plane? 

The screw top you find on some jars is designed like a screw, which is a type of inclined plane.

What I'll Learn 

  • I CAN make various inclined planes using K’NEX.
  • I CAN relate inclined planes to triangles.
  • I CAN make a coordinate plane using two number lines.
  • I CAN use a protractor to measure angles.
