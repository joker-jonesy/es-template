The Challenge - IQ Key: Compare Machines

The Challenge

Your challenge is to build, test, and evaluate a crane machine. Then you will see if you can improve the design. You will start by building the basic crane and calculating the work done when it lifts a load. Then you will see if you can improve the crane design and calculate the work done by your modified crane for comparison.

What I'll Learn 

  • I CAN test the performance of a machine and evaluate its performance mathematically.
  • I CAN collect data from experiments.
  • I CAN multiply fractions and mixed numbers to solve problems.
  • I CAN show my work using equations and visual fraction models.
  • I CAN modify the basic design of a machine in an attempt to improve its performance.
  • I CAN compare the performance of two vehicle designs by including mathematical comparisons.
