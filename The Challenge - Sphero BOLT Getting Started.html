The Challenge - Sphero: Germs Everywhere 

The Challenge

*This project starter requires the specific capabilities of the Sphero BOLT robot. 

Your challenge is to program Sphero BOLT to simulate the spread of an infectious disease. Learn how to program the BOLT robot to communicate with other BOLT robots to “spread a disease”, and use a line plot to track the spread of the disease. You will conduct at least three trials so you can find the average number of robots infected in specific time intervals. You will also do calculations to find the total number of infected robots at each time interval.

What I'll Learn 

  • I CAN create and edit programs using block coding.
  • I CAN program BOLT to relay messages to other BOLT robots using infrared (IR) communication.
  • I CAN create a line plot to record data in fractions.
  • I CAN use data from a line plot to solve mathematical equations involving fractions.

Stuff I'll Need

  • Multiple Sphero BOLTs
  • iPads or other mobile devices with Sphero EDU app (one for each robot) 
  • Stopwatch or Timer
