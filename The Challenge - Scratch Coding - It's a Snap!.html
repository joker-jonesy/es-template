The Challenge - Scratch: Coding - Check Your Work with Computer Programs

The Challenge

Your challenge is to create your own animation with Scratch. If you are new to Scratch, you will complete the Getting Started Tutorial. Then you can program your animation. Once you are comfortable with Scratch, use the standard algorithm to solve multi-digit multiplication and then check your answers using a simple Scratch program. 

What I'll Learn 

  • I CAN create an animation using block coding.
  • I CAN multiply multi-digit whole numbers using the standard algorithm to determine the product of various scenarios.
  • I CAN use code to solve multi-digit multiplication problems.

Stuff I'll Need

  • Scratch Website
  • Scratch Software
    • Mac: in your dock
    • PC: All Applications folder
