What is VEXcode VR?

VEXcode is a web application that allows you to program a robot to:

  • navigate a virtual, or online, maze
  • draw
  • use a magnet to pick up and move objects

How Do You Use VEXcode VR?

Project Editor

When you open VEXcode VR, you will see the Project Editor. This is where you program your robot and can watch a simulation of your program.

Code Blocks

VEXcode VR uses blocks instead of symbols, numbers, and letters.

Each block represents a line of code. These blocks of code snap together like puzzle pieces.

By connecting blocks of code, you create a computer program.

Blocks of code tell the robot what it should do. VEXcode organizes blocks using color. Each color represents a different set of block commands.

Scripts Area

To program your robot, you will drag the code blocks into the scripts area of the workspace.

Note: Along the top right of the Project Editor, you will find your program control options. To open the simulator tool, click on the Playground icon, which is to the left of the start button.

Explore On Your Own

VEXcode includes step-by-step tutorials. Use the tutorials to help explore the VEXcode project editor and to help answer your questions while you're working on your project.

You can get to this section by clicking on the lightbulb icon in the toolbar along the top of the Project Editor.
