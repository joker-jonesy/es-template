  • If you do not have a Hungry Alligator, build one first.

Load WeDo Blocks into Scratch

To program WeDo with Scratch, you need special blocks to control the motor and sensors.

  1. Open Scratch.
  2. Select Create.
  3. Click on More Blocks under the Scripts tab.

If you do not see the Lego WeDo blocks, follow these steps:

  1. Click the Add an Extension button.
  2. Click on the WeDo button.

  

Still no blocks? Go to Software Check in the What You Should Know section and click on "Installing the Scratch Device Plugin".

Program Your Hungry Alligator

Program your Hungry Alligator using Scratch. Start with something simple. Look at the Scratch sample code shown here. It programs the alligator to open and close its mouth. Try to create this code yourself:

Create New Behavior

Now use Scratch to create new behaviors for your alligator.

Here are some ideas:

  • Change the sound your alligator makes.
  • Create two sounds: one for opening the mouth and one for closing the mouth.
  • Change the length of time the alligator’s mouth is open or closed.
  • Change the activation key from 'a' to the space bar or any other key.
  • Program two keys to control the alligator: use one key to open the mouth and a different key to close the mouth.
  • Create an on-screen animation to show a Scratch Sprite opening and closing its mouth at the same time your alligator's mouth opens and closes.
  • Use Scratch to control the distance sensor for opening and closing the alligator’s mouth.
