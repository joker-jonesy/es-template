Patterns All Around

We see patterns all around us. In nature, in buildings, and in art. Patterns repeat the same shape over and over. 

  

What is a Tessellation?

A tessellation is a special kind of pattern. Tessellations are flat surfaces with one or more shapes that repeat without any gaps.

Tessellations can be simple, like a checkerboard, or very complex. All of these patterns are tessellations. Can you find the repeating shapes in each pattern?

Tessellations in Nature

The idea of tessellation patterns is not man-made. In fact, it can be found all around us in nature. 

  

About VEXcode VR

VEXcode is a web application. With VEXcode VR, you can program a robot to:

  • navigate a virtual, or online, maze
  • draw in several colors
  • use a magnet to pick up and move objects

Coding with Blocks

VEXcode VR uses blocks instead of symbols, numbers, and letters.

Each block represents a line of code. These blocks of code snap together like puzzle pieces. 

By connecting blocks of code, you create a computer program.

There are different kinds of blocks. Each type comes in a different color.

Project Editor

When you open VEXcode, you will see the Project Editor. This is where you program your robot and can watch a simulation of your program.

A simulation will show you what the robot would do in real life if you had it in front of you.

Code Blocks

Code blocks tell the robot what it should do.

VEXcode organizes blocks using color. Each color represents a different set of block commands.

In this challenge, you will use the Drivetrain, Looks, and Control code blocks.

Scripts Area

To program your robot, you will drag the code blocks into the scripts area of the workspace.
