Understanding File Formats and Extensions

Have you ever tried to open a file in a program and get a message that the file type is not supported by that program? It may be because it is not in the correct file format. To find out what format the file is in - look at the file extension. File extensions usually come after the saved name and a dot. For example: If you are trying to open a Word document with the file extension .doc in Excel, you will not be able to because Excel reads files with extensions .xls.

You can usually tell what program you need by looking at the file’s file extension. This is a Word document because it has the file extension .doc. My_journal.doc

Here are some common file extensions & formats:

  • doc (Microsoft Word - Document)
  • .xls (Microsoft Excel)
  • .pdf (Portable Document Format – used by Adobe Reader)

Common image files:

  • png - Portable Network Graphics
  • jpeg - Joint Photographic Experts Group
  • gif - Graphic Interchange Format

Common audio/video files:

  • .mp3 – used for audio
  • .mp4 – used for video and audio

In Tinkercad you will open a file with the extension - .svg (Scalable Vector Graphics). Watch the video CS Basics: File Extensions to learn more about file extensions.

What is a Stamp

What is a stamp, sometimes referred to as a rubber stamp or ink stamp?

Your challenge is to create your own stamp, sometimes referred to as a rubber stamp or ink stamp. A stamp is “a handheld device for inking and imprinting a message or design on a surface."

The rubber stamp was invented in the 1800s. Several men claimed they invented it, but no one is really sure. Stamping with wooden blocks is even older. Some are as old as 100 A.D. You can learn more about the history of the stamp here.

Stamps come in many shapes, sizes, and designs and are used in many ways.

  • Post Offices use them to stamp parcels.
  • Teachers use them to stamp messages on student work.
  • Medical companies will use them to mark the packages they send to hospitals and clinics.
  • Many people use them for crafting.

Career Connection and Real-World Application

3D Shapes in Careers

There are a lot of ways that 3D shapes are used in the real world. Many careers involve 3D modeling. Some of these careers include video game designers, animators, and architectural designer. All of these careers involve creating 3D models in a computer program. 

Video game designers and animators use their skills to create 3D designs that are used in games, cartoons, and movies. They work at animation studios. 

Architectural designers use computer programs to create 3D models of buildings before they are built in real life. They use their computer models to plan the best designs for their clients.
