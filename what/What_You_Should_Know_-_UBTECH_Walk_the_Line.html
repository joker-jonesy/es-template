  • Inches - a unit of length, there are 12 inches in a foot
  • Feet -  a unit of length
  • Length - how far it is from one thing to another
  • Line Plot - a way to show data using a number line
  • Yard - a unit of length, it is equal to 3 feet or 12 inches

Measuring Lengths and Distances 

When we measure how long something is or how far something moves, we are measuring length. We use different units of measurement to measure lengths depending on how small or how large the distance is, some common measurements include centimeters (cm), inches (in), feet (ft), meters (m), or yards (y). A ruler is a handheld measurement tool that commonly shows the markings for some of these units. The common ruler below shows inches on one side and centimeters on the other. 

A ruler also measures fractions of length. The ruler here is marked with whole number measurements but also parts of whole numbers including ¼ inches and ½ inches. 

Line Plots

A line plot is a way for you to show the data that you have collected in an easy to read way. A line plot runs along a number line and x’s or check marks show how many times something happens. In the example, below, you can see the number of students who have pencils. One student has 6 pencils and 4 students don’t have any pencils at all.

Coding 

Coding is a way to give instructions to a computer. The code or program are instructions for a computer - it tells the computer exactly what to do. Code needs to be precise for a computer to successfully follow it. For example, your code can’t simply say move- the computer needs to know how to move, how long to move, and when to stop. The code instructions must also be written in the right order. Does the robot need to go forward for three seconds and then turn? Or turn and then go forward for three seconds?

There are different ways to code a computer called programming languages. UBTECH's programming language uses code blocks to create a program. The code blocks tell the computer what to do. These blocks can be dragged and dropped into a workspace and they click together just like puzzle pieces to create list of things the computer will do.

Career Connection and Real-World Application

Scientists and Teachers

Line plots can help to organize data for meteorologists or other scientists. They can use them to display data like the amount of rain that falls every day or the temperatures every day at a certain time. Line plots help make data more visual, this helps scientists more easily identify trends and other important information. 

Teachers also use line plots to graph useful data to help inform decisions. A line plot showing student test data could help a teacher to better understand the concepts that the students understand very well and quickly determine what concepts might need additional teaching.
