Writing Code

Like people, computers have their own language they use to communicate. Computers use a written language called source code  to work, and humans can tell computers to do things by writing source code. Source code is a language that contains numbers, symbols, and words. In Code Combat players are able to explore, learn, and practice three different script languages including the popular Python and JavaScript, and an experimental code called CoffeeScript.

Methods

Methods are actions your hero can do with their gear. Your hero has feet so he can move up, down, right, or left. At the end of each method there are parenthesis where an argument can go, shown in the example below. Methods for Java Script are similar to Python except they have a semicolon at the end of each line, hero.moveLeft(); moves the hero left, hero.moveUp(); moves the hero up etc. 

Arguments and Strings

Arguments in Java script language tell the computer more specific details about a method or action. For example, you could tell the computer to move the hero right with hero.moveRight(); or you could use an argument to be more specific about how far you want your hero to move right. You could tell the computer to move farther. If you write hero.moveRight(4);, your hero will move to the right 4 times.

A string is a piece of text information that is unique to the situation. To tell the computer about a string, surround text with quotation marks.

You can combine arguments and strings by adding quotation marks. For example, you could tell the computer to have the hero attack with hero.attack(enemy) but you must add an argument and string to give your hero specific information about the situation, hero.attack("Treg");

Loops

Sometimes a certain piece of code needs to be repeated over and over. You can use a "while-true" loop and tell the computer to repeat an action however many times you want.

All "while-true" loops in Java script must start with a bracket { and end with a bracket } and each of the steps that you want to be repeated forever while the scenario is true must be indented 4 spaces under the  while (true) words.
