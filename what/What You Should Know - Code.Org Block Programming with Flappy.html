Important Vocabulary

  • Computer Code - instructions that tell a computer what to do
  • Command - a single instruction that tells a computer what to do
  • Event - an action that causes something to happen
  • Workspace - the place where you do your work
  • Multiple - a number that can be divided by another number without a remainder

Computer Code Is Everywhere

Computer code is integrated into more than you may realize.

Smart phones, electronic toys, and cars all need computer code to function.

What is computer code? Computer code is a set of instructions that tells an electronic device what to do. Each line of code is a single instruction or command. Computers and other electronic devices seem smart, but this is only because a human (e.g. computer programmer, software engineer, developer) wrote a set of instructions for them.

How does computer code work? How does a game character know what to do when you push a button? A signal from the keyboard or controller travels through the wires. The signal is read by the computer and the code tells the character to move. The computer program indicates what needs to happen in the game depending on the buttons you push.

Block Programming

Learning to code can be fun with block programming. The code involves dragging and dropping blocks that represent commands for the computer. Each block represents real code. And fits together with other parts of the code like puzzle pieces.

What Are Multiples?

A multiple is a number that can be divided by another number without a remainder. The numbers 10, 20, 35, and 40 are all multiples of 5 because each can be divided by 5 without a remainder.

Career Connection and Real-World Application

Video Game Developer

Video Game Developers write code for video games. They combine creativity, art, math, and programming skills to create video games for education or entertainment.

Web Designer

Web Designers write code to build a variety of websites for use on tablets, phones, and laptops. They consider the design, the user experience, and the purpose to create the websites that we all interact with on a daily basis.
