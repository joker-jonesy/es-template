Important Vocabulary

  • Computer Programs - Computer programs are the instructions that tell electronic devices what to do
  • Multi-digit - when a number has more than one digit
  • Standard Algorithm - a process used to multiply in parts or by using partial products
  • Whole Number - a number that does not contain fractions or decimals
  • Product - the answer to a multiplication problem in mathematics
  • Factors - the numbers being multiplied

How Stuff Works

Many things "know" how to behave.

A robotic dog "knows" how to bark and wag its tail. Phones "know" to call or text depending on which button you touch. Even your house "knows" when to turn on the heat or air conditioning.

How do machines, toys, and electronic devices know what to do?

Inside each of them are small electronic parts that contain information. The information is known as computer programs.

Computer programs are the instructions that tell electronic devices what to do. Computer programs are written by software engineers.

Talking to a Computer

How do software engineers communicate with computers?

We communicate with computers using symbols and numbers.

This is similar to how we use language to talk to each other, but this is a language that the computer understands.

The instructions are written in lines of code. Each line of code tells the computer to do something. The lines of code in this image tell the computer to print “Hello, World!”

Multiplying Using the Standard Algorithm

The standard algorithm of multiplication is a process used to multiply in parts or by using partial products. This process is used as a way to multiply numbers no matter how large or small.

  1. Line up the numbers by place value (this is very important to find the appropriate solution).
  2. Multiply the top number by the ones place in the bottom number (regroup if needed).
  3. Put zero as a placeholder on the second row of your answer.
  4. Multiply the top number by the tens place in the bottom number (regroup if needed).
  5. Add to get your final answer (regroup if needed).

Coding with Blocks

Scratch uses blocks instead of symbols, numbers, and letters.

Each block represents a line of code. These blocks of code snap together like puzzle pieces.

By connecting blocks of code, you create a computer program.

There are different kinds of blocks. Each type comes in a different color. 

Career Connection and Real-World Application

Malware Analyst

A career known for assisting with debugging computer code is a malware analyst. Malware analysts investigate, program, and reverse engineer software to rid it of errors.

Multimedia Animators/Artists

Multimedia animators and artists design visual and special effects using computer software or written code to create their content. A multimedia animator and artist may also be known for creating animations, games, and visual images using videos, computers, film, coding, or other media.
