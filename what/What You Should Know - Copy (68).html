Understanding File Formats and Extensions

Have you ever tried to open a file in a program and get a message that the file type is not supported by that program? It may be because it is not in the correct file format. To find out what format the file is in - look at the file extension. File extensions usually come after the saved name and a dot.

If you are trying to open a file with a file extension that is not compatible with the program it will not work. For example, a Word document with the file extension .doc will not open in the Excel program because Excel reads files with extensions .xls.

What is a Stamp?

A stamp is “a handheld device for inking and imprinting a message or design on a surface."

The rubber stamp was invented in the 1800s.  Several men claimed they invented it, but no one is really sure.  Stamping with wooden blocks is even older.  Some are as old as 100 A.D. You can learn more about the history of the stamp here. 

Businesses use stamps for many reasons. Rubber stamps are sometimes used by the Post Office when shipping things, they stamp a letter or a package to keep track of the day it was received at the Post Office. Some businesses use stamps to stamp their address, logo, or other business information onto a page.

Geometry

Geometry is the study of the size, shape, angles, and dimensions of objects. A mathematician that works in the field of geometry is called a geometer. A geometer can use some of these important geometry terms to describe objects: 

Point

A specific position in space, usually represented by a dot and named with a letter. An example would be point A.

Line Segment

A section of a line that has a point on either end. Line segments are labeled by the points at either end. For example, line segment AB.

Angle (right, acute, obtuse)

The space between two lines or line segments that is measured in degrees. Acute angles are less than 90°. Right angles are exactly 90°. Obtuse angles are greater than 90°. 

Perpendicular Lines

Perpendicular lines form a right angle and make a “T”.

Parallel Lines

Lines lines never intersect and have the same distance between them. These lines never touch. 

Symmetry

Exactly the same on either side of an intersecting line. The two sides are mirror images of each other.
