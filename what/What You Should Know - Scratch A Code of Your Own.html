What You Should Know - Scratch: Program to Solve Word Problems

Important Vocabulary

  • Sprite - a character you use or design on Scratch
  • Variable - a value that can be inserted into coding and changed as needed
  • Script - a series of coding commands that cause something to happen
  • Tutorial - instructions with examples, often presented with video or pictures
  • Operator Blocks - Green Scratch blocks that are used to include math equations into your code

Interactive Programs and Animations

Making a program or animation interactive means that it does different things based on what you do, and the instructions you give it.

For example, your alarm clock goes off at the same time every morning because you program it to do that. Sometimes you may want it to behave differently. If you hit the "snooze" button to sleep in, the alarm sounds at a later time, and yet, you do not have to reset the alarm.

If you want a character in a game to move, you click a mouse or push a button. You do not have to rewrite the program every time. For your animation, you can choose to make it interactive. For example, you can program a character to speak or move only when it is clicked. When you run the program, you can choose to click on the character once, many times, or not at all.

Math and Scratch

You can use your Scratch animation to help you practice multiplication and division. For example, if you want your character to walk forward 10 steps, you can simply use the Motion block to go 10 steps, or you can insert the multiplication Operator block into the Motion block to have your character go 2 x 5 steps, which also comes out to 10. 

You can use those same Operator blocks to create and solve word problems. There are Looks blocks that command your character to “say” or “think” something. Using one of these Looks blocks, you can program one character to say a word problem and have another character solve the problem. If you put a multiplication or division Operator block into the Looks block, Scratch will solve the problem for you and your character will “say” the answer.

Career Connection and Real-World Application

Coding in Careers

There are a lot of jobs that use coding. Coding is used in computer programming, game creation, and website development. The codes used on Scratch are called block codes. Learning to create scripts using Scratch’s block codes is a good way to begin to understand how coding works. 

Robotic engineers often use block coding like the codes used in Scratch scripts. Block codes are inserted in different orders to tell robots what actions to perform.
