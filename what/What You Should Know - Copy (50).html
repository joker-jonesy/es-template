Game Controllers and Sensors

Game controllers and sensors connect you to your computer and let you communicate with it. By pressing buttons or moving the controller, you change the action on the computer or TV screen. Lego WeDo sensors do the same thing.

Software Check

Do you need the Scratch – WeDo plugin?  

  1. Open the Scratch website. Click here.
  2. Select Create.
  3. Select More Blocks.
  4. Select Add an Extention.
  5. Choose LEGO Wedo 1.0.

If you see the following message and a red light next to the WeDo blocks, you will need to install the WeDo plugin by following the Installing the Scratch Device Plugin instructions. Check with your Facilitator before installing the WeDo plugin.

If you see no message and the light beside the WeDo blocks is yellow or green, you can continue to the Do It! section.

 

Installing the Scratch Device Plugin

Install the Scratch Device Extension for your computer:

  1. To install the Scratch Device Extension to your computer, click here.
  2. Choose the correct download link for your browser and operating system under Download the Plugin.
    For example, select Windows (Chrome) if you are using Chrome as your browser on a Windows computer.
  3. Select where you would like to save the file and click Save.
  4. When it finishes downloading, go to the saved file and double-click the icon.
  5. Choose Run and then Yes/Allow if prompted.
  6. If you are using Chrome as your browser, continue on below and complete the next set of steps. If you are using a different browser, repeat the steps in the Software Check section above to make sure the extension is working properly. 

                                  

Install the Scratch Device Plugin for Chrome:

  1. To install the Scratch Device plugin to your browser, click here.
  2. Select the separate plugin helper link for Mac (Chrome) or Windows (Chrome).
    You will be brought to the Scratch Device Plugin Helper page.
  3. Choose Add to Chrome.
  4. Select Add extension.
  5. Repeat the steps in the Software Check section above section to make sure the extension and plugin are working properly.
