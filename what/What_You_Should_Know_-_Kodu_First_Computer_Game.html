Important Vocabulary

  • Polygon - a closed shape formed by connected straight lines
  • Two-dimensional Shape - a flat, closed shape that has length and width
  • Three-dimensional Shape - an object that has length, width, and height
  • Area - the size of a surface
  • Code - instructions for a computer
  • Aerial View - pictures that are taken from above an object or location
  • Terrain- the land or ground

Writing a Computer Program

When you make a computer game, you are actually writing a computer program. Your program tells the computer how your game is played using instructions called code. In some computer languages, you write code by typing. In Kodu, writing code is as easy as clicking your mouse.

Each line in this picture is a line of code. Each line acts as a rule or command that tells your computer how your game is played. A rule is a line of When & Do commands.  (When ____ happens, the object should do ____.)

Finding Area with Tiling and Multiplication

Area is the total space that fills in a shape. You can find the area of a shape using different methods. One way to find the area of a shape is to use multiplication.

To find the area of a shape, we can multiply the length by the width of a rectangle or square.

A = L x W

If the length of a rectangle is 4ft and the width is 3ft, we can multiply these dimensions to find the area.

A = L x W

A = 4 x 3

A = 12 ft

Career Connection and Real-World Application

Architect

Architects design buildings, they need to figure out the area of buildings and rooms within them so they can design buildings and structures that we can use. If they design a stadium, they need to figure out how much area they need for a field and how much area they need for seats.

Carpenter

Carpenters work to build the plans of architects. If architects design a home to be a certain size, carpenters will use math to figure out the different amounts of materials used for the floors, walls, and ceiling like carpet, paint, curtains, or even specific appliances, like stoves and toilets.
