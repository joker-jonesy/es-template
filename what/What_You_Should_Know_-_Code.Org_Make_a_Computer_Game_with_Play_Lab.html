Important Vocabulary

  • Block-Based Programming Language - any programming language that lets users create programs by using “blocks” rather than writing code using text
  • Command - a single instruction that tells an electronic device what to do
  • Event - an action that causes something to happen
  • Pixel - short for "picture element" - the basic unit of a digital image, typically a tiny square or dot that contains a single point of color of a larger image
  • Loop - the action of doing something over and over again
  • Perimeter - the distance around a flat shape

Computer Code Is Everywhere

Computer code is a part of more products than you might think. Smart phones, electronic toys, and cars all need computer code to work.

What is computer code? Computer code is a set of instructions that tells an electronic device what to do. Each line of code is a single instruction or command. Computers and other electronic devices seem smart, but this is only because a human (e.g. computer programmer, software engineer, developer) wrote a set of instructions for them. 

How does computer code work? How does a game actor know what to do when you push a button? A signal from the keyboard or controller travels through the wires. The signal is read by the computer and the code tells the actor to move. The computer program indicates what needs to happen in the game depending upon the buttons you push.

Block Programming

Learning to code can be fun with block programming. Each "block" represents a part of a command. A block fits together with other blocks like a jigsaw puzzle. When you drag, drop, and attach blocks, you are putting together your code. Each block represents real code.

What Is a Pixel?

A pixel is one point on a picture. Think of a pixel as a small dot or square of color on our computer, movie, or television screen. When pixels are put together, images are formed. If an image is 200 pixels, that means the image is made up of 200 little points of color.

What Is Perimeter?

Perimeter is the distance around an object. For rectangular, flat objects, there are two ways to calculate or find the perimeter.

  1. Add the length of each side. The perimeter is the total length or sum of all sides.
  2. Multiply the length times two, multiply the width times two, and then add the products together.
