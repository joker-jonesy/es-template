Do It!

Your challenge is to program your first VEX robot to create a tessellation. 

Explore the Tutorials

VEXcode includes step-by-step tutorials. Use the tutorials to help explore the VEXcode project editor and to help answer your questions while you're working on your project.

You can get to this section by clicking on the lightbulb icon in the toolbar along the top of the Project Editor.

Watch the following tutorials:

  • Getting Started
  • Moving and Removing Blocks
  • Driving Forward and Backward
  • Turning
  • Using Loops

These tutorials will teach you the code blocks you need to use to create a tessellation.

Open the Playground

The Playground lets you see a simulation of your program in action.

  1. Open VEXcode VR.
  2. Open the Playground simulator.
  3. Open the Art Canvas playground.

Sketch Your Tessellation

Before programming your robot, plan your tessellation. 

What shapes will make up your tessellation? How will your robot turn to create the sides of each shape? How will each shape connect or overlap with the other shapes next to it? 

Program Your Robot

Now you will use the code blocks to control the pen and draw your tessellation.

Drag blocks into the scripts area to form your program.

Controlling the Pen

Use the purple code blocks in the Looks section to set the robot pen color and and move the pen down so it can write on the canvas.

Creating a Shape

Use blue code blocks in the Drivetrain section to move the robot forward, backward, and turn.

Creating a Tessellation

Use the orange repeat, or loop, code blocks in the Control section to repeat code blocks to create multiple shapes.

Test and Debug

Once you have written your initial program, it is time to test it out.

You can run your program by clicking on the play button inside the Playground simulator. 

Sometimes your program doesn't work the way you intended. Debugging, or finding mistakes and fixing them, is a normal part of software engineering. 

If your program doesn't work at first, go through and find where you may need to add, change, or remove code blocks in your program. 

Reset the playground simulator and run your program again. Continue this process until your robot creates your tessellation.
