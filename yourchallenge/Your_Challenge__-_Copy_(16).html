Scratch: A Code of Your Own

Your Challenge

Computer games are popular because people like to be a part of the action. It is fun to make things happen and try to reach a goal.

Your challenge is to use Scratch to program an interactive computer game. You can add to an existing game, or plan and create your own game "from scratch." 

What You'll Learn

  • Students will learn about improving games with code.
  • Students will learn about creating their own games by coding. 

Stuff You'll Need

You will need one of the following:

  • Scratch website
  • Scratch software
    • Mac: In your dock
    • PC: All Applications folder
