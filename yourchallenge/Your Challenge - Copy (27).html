Code Combat: Introduction to Python

Your Challenge

In this Liftoff, you will learn how to control a character in a computer game. Your challenge is to use Python Script coding language to program your hero in Code Combat to defeat evil and collect gems.  

**The first 21 levels in the Kithgard Dungeon are free to all users, but any levels beyond the first must be purchased. You can move into the Forest and complete free levels in that world. If you would like to access more levels and learn more coding skills, ask your facilitator to purchase a premium class subscription on the Code Combat website!  

What You'll Learn

  • Students will learn about writing code.
  • Students will learn about using the python script language.

Stuff You'll Need
