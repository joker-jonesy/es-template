Do It! UKIT Beginner: Walk the Line

 The Challenge

Your challenge is to build a robotic car that is able to move forward using the UKIT Beginner kit. You will use your measuring skills to measure the distances your robot car travels and then you will create a line plot of your data.

 Project Steps

  1. Build Your Robot

  2. Code Your Robot

  3. Measure Your Distance

  4. Collect Data and Create a Line Plot

Code Your Robot

Now that your robot car is built, it’s time to program or code your robot car. Upload the UKIT EDU App to your phone if you have not already done that.

Step 1: Select My Project, then select the plus sign to add a new project. 

Step 2: Turn on your robot car and click the Bluetooth icon to connect. The app will search for your robot car. Make sure your robot car is turned on! 

Step 3: Select your robot from the list. If there is more than one robot listed, choose the one with the strongest signal strength. Connect to your robot car. Check that you see 3 servos associated with the robot you selected. 

Step 4: Select Moves. Drag and drop your commands into the workspace. 

Step 5: Set servo ID-1 to rotate 90 degrees. Leave servos ID-3 and ID-4 set to 0 degrees.  

Step 6: Select servos  ID-1, ID-3, and ID-4 and change the duration to 2000 ms. 

Step 7: Select Control. Scroll until you see the command Wait. Drag and drop it into the program. 

Step 8: Change the wait time to 4000 ms. This tells the robot car to wait 4 seconds and will give you time to make sure your ruler is in place. 

Step 9: Select Moves to adjust wheel motion. Select rotate servo block and change servo ID-3 to clockwise and slow. Change servo ID-4 to counterclockwise and slow. Do nothing to servo ID-1. 

Stop! Your next task is to run your robot car! You will measure how far the robot car can travel using your ruler and record your data. Then, try to change the program to have the robot travel different distances. 

Measure Your Distance

Using the program you just created, run your robot car and measure the distance the robot car travels. Record the distance in the data table below. Start the robotic car at the zero point of your ruler. Line up the front axle with the 0 on the ruler.

The program tells the robot car to move in a straight line. You may see the robot move a little bit as it finds the correct orientation. The 4 second wait that you programmed into the robot car gives you time to make sure the robot is lined up with the ruler correctly before it begins to move. 

Once you have measured the distance the robot traveled, program the robot to move twice as far. When you have successfully gotten the robot to travel twice as far, take a screenshot of the program you altered and insert it into a table like the one below and save for your project submission. If there is time, program the robot to go half as far or three times as far.

Collect Data and Create a Line Plot

Now we will alter the program by adding to the original program a random number generator. This program will randomly generate a distance that the car will travel without you knowing what that distance will be ahead of time. You will run the robot car ten times. Each time the robot car runs, you will measure how far it traveled and record it in your data table. When you have ten measurements, you will organize them by creating a line plot. You will upload your data table and line plot for your project submission.
