The Challenge - Vernier Science: Scientific Method

The Challenge

Your challenge is to learn how to do an experiment using the scientific method. You will practice the scientific method by doing a class temperature experiment using the Vernier Go!Temp sensors and the Graphical Analysis software to see who has the hottest hand. Then, redo the experiment using place value relationships to set new time and sample intervals. 

What I'll Learn 

  • I CAN use the Go!Temp probe to collect data.
  • I CAN utilize graphical analysis software to display my data.
  • I CAN use the scientific method to do a science experiment.
  • I CAN use place value relationships in multi-digit numbers.
  • I CAN recognize that a digit in one place represents 10 times as much as it represents in the place to its right and110 of what it represents in the place to its left.
