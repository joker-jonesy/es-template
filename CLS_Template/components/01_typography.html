Heading Levels

Best Practices

Heading levels help to organize content and create hierarchical relationships between pieces of content. Heading levels here can be used in the same way you would use heading levels in a Word document: begin with Heading Level 1, sub-headings beneath that should use Heading Level 2, sub-sub-headings beneath Heading Level 2, should use Heading Level 3 and so forth.

Every time you create a new content page (Create a File), you'll need to begin the page with a Heading Level 1. For accessibility, each page should only have one Heading Level 1. All other heading levels can fall underneath Heading Level 1 and should never skip a level. For example, a Heading Level 3 should never fall immediately beneath a Heading Level 1 without a Heading Level 2 in between.

Paragraph text

Best Practices

Paragraph text is plain text that should be used for the majority of your content. This content will always appear as a single column of text. It should be used for complete sentences and broken into paragraphs using proper grammar and prose styling.

Paragraph text.

Lead text (.lead)

Best Practices

Lead text is used to begin a new page of content or section of content. Typically, lead text would only follow a heading. Lead text can be used to ask the learner a rhetorical question, make a critical point about the content that will follow, or provide an overview statement about the content that will follow.

Emphasis text (.emphasis)

Best Practices

Emphasis text is used within content to make a critical point or emphasize particularly important information without disrupting the flow of content. Emphasis text should be used sparingly. Consider establishing your own rules or patterns for incorporating emphasis text, for example, you might always provide a summarizing statement at the end of a section using emphasis text OR you might consider using emphasis text to present key idea, then elaborate on the idea using paragraph text.

Instruction text (.instruction)

Best Practices

Instructions should appear before any dynamic element. You can use generic phrasing for instructions, for example 'Select each item to learn more', or include instruction that are more specific to the interaction you are building.

You may also consider using instruction text for more general instructions, for example you may include instructions at the bottom of each page for what the learner can expect next, for example, 'On the next page, we will learn about' or 'Up next, you will be asked to complete Quiz 1.'

Image Blockquote

Best Practices

An image blockquote should be used when the quotation is the subject of all the content that will follow. As an example, consider if you were unpacking the meaning behind the Dale Carnegie quotation, 'Inaction breeds doubt and fear. Action breeds confidence and courage. If you want to conquer fear, do not sit home and think about it. Go out and get busy.' This quotation would work well as an image blockquote starting off the section of content explaining it.
