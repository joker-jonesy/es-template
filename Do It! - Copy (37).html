Do It!

Your challenge is to build an inclined plane machine so you can show how inclined planes make work easier.

Building Inclined Planes with K’nex

Steps to complete this challenge:

  1. Make sure you know how to use the K'nex parts.  
  2. Start with a basic model as a warm up.
  3. Choose an advanced machine to complete, or create your own machine that incorporates an inclined plane somewhere in the design.

  4. For all of your projects, make sure you label the parts your machine and explain how your machine helps do work. Go to the What You Should Know section if you need to review these concepts.

Basic Models

For this Liftoff Challenge, build one or more of the basic inclined plane models from the instruction manual. Take a picture and label its parts. What type of inclined plane does it use? Explain in your own words how your model helps make work easier.
